# 技能加固总览

## 一、加固目标

### 1.1 加固原则

```
1. 实战导向：每个技能都有实战案例
2. 可执行性：每个技能都有具体步骤
3. 可复制性：每个技能都能快速复用
4. 关联性：技能之间有清晰的关联
5. 可迭代：支持持续优化
```

### 1.2 加固标准

```
□ 有明确的使用场景
□ 有具体的操作步骤
□ 有实战案例
□ 有检查清单
□ 有常见问题
□ 有关联技能
```

## 五、加固检查清单

### 5.1 内容检查

```
□ 每个技能都有实战案例
□ 每个技能都有操作步骤
□ 每个技能都有检查清单
□ 每个技能都有关联技能
□ 内容准确、无过时
```

### 5.2 结构检查

```
□ 目录结构清晰
□ 文件命名规范
□ 交叉引用正确
□ 无重复内容
□ 无冲突内容
```

### 5.3 功能检查

```
□ 触发词正常
□ 技能可执行
□ 案例可复用
□ 检查清单实用
□ 关联图清晰
```
