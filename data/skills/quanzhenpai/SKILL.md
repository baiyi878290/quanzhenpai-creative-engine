## 工作流程

```
用户输入
  │
  ├─ 剧本改编？ → 加载 references/script-adaptation.md
  │    ├─ 网络小说 → 压缩→分场→剧本
  │    ├─ 漫画 → 分镜→剧本
  │    └─ 真实事件 → 采访→大纲→剧本
  │
  ├─ 分镜创作？ → 加载 references/storyboard-creation.md
  │    ├─ 实拍分镜 → 景别+运镜+布光
  │    ├─ AI分镜 → 提示词+一致性
  │    └─ Agent Prompt → prompts/storyboard-agent-prompt.md
  │         ├─ StoryboardMaster System Prompt (v1.9 完整语法)
  │         ├─ 双轨分镜输出 (实拍+AI)
  │         ├─ 15列标准分镜表 + AI生成参数
  │         └─ 防幻觉机制 + 质量自检清单
  │
  ├─ 图片提示词？ → 加载 references/image-prompt.md
  │    ├─ 角色设定 → 多角度参考表
  │    ├─ 场景设定 → 光影+氛围
  │    └─ 分镜草图 → 构图+情绪
  │
  └─ 视频提示词？ → 加载 references/seedance-prompt.md
       ├─ 文生视频 → 场景描述+运镜
       └─ 图生视频 → 参考图+动作描述
```

## 输出格式

### 剧本改编输出
```markdown
# 《剧名》竖屏短剧剧本
## 基本信息
- 类型：[甜宠/逆袭/虐恋/复仇/悬疑]
- 集数：[X]集 × [Y]分钟
- 目标平台：[抖音/快手/微短剧]

## 人物小传
## 分集大纲
## 详细剧本（分场）
```

### 分镜脚本输出
```markdown
# 分镜脚本

| 镜号 | 景别 | 运镜 | 画面描述 | 台词/音效 | 时长 | 备注 |
|------|------|------|----------|-----------|------|------|
| S001 | 特写 | 推 | ... | ... | 3s | ... |
```

### 提示词输出
```
[主体描述], [光影], [氛围], [风格], [技术参数]
```

## 风格适配

支持12类从业者风格：
1. 导演风格 - 视听语言、分镜节奏
2. 编剧风格 - 剧本结构、对白设计
3. 摄影指导风格 - 光影构图、镜头运动
4. 剪辑师风格 - 节奏把控、蒙太奇
5. 调色师风格 - 色彩方案、氛围营造
6. 制片人风格 - 预算、周期、资源
7. 网文作者风格 - 爽点密度、更新节奏
8. 编辑风格 - 选题判断、市场定位
9. 投流优化师风格 - 短剧投流、ROI分析
10. IP运营风格 - 版权开发、跨媒介
11. 学术研究风格 - 理论分析、文献支撑
12. 创业者风格 - 商业落地、资源整合

## 五、剧本→提示词自动化

### 工作流程

```
剧本/分镜脚本
    │
    ├─ 角色信息提取 → 角色设定板提示词
    │    ├─ 左右分割式设定板
    │    ├─ 九宫格多角度
    │    └─ 六表情参考
    │
    └─ 分镜信息提取 → 分镜视频提示词
         ├─ 每镜头提示词
         ├─ 场景设定提示词
         └─ 一致性控制
```

### 触发词

| 触发词 | 功能 |
|--------|------|
| 生成角色设定板 | 从剧本提取角色，生成设定板提示词 |
| 生成分镜提示词 | 从分镜脚本生成每镜头提示词 |
| 批量生成提示词 | 一次性生成所有角色+所有镜头 |
| 角色参考图 | 生成Seedance用的角色参考图 |

### 使用示例

```
用户：帮我从这个剧本生成角色设定板
AI：[读取剧本] → [提取角色信息] → [生成设定板提示词]

用户：生成第1集的分镜提示词
AI：[读取分镜脚本] → [提取每镜头信息] → [生成视频提示词]

用户：批量生成所有提示词
AI：[读取剧本+分镜] → [提取所有信息] → [生成所有提示词]
```

## 六、一键成片工作流

### 完整流程

```
剧本/小说
    │
    ├─ Phase 1: 信息提取（1轮对话）
    │    ├─ 角色卡
    │    ├─ 场景卡
    │    ├─ 道具卡
    │    └─ 分镜脚本
    │
    ├─ Phase 2: 设定板生成（批量输出）
    │    ├─ 角色设定板提示词 → GPT-Image-2 → 角色参考图
    │    ├─ 场景设定板提示词 → GPT-Image-2 → 场景参考图
    │    ├─ 道具设定板提示词 → GPT-Image-2 → 道具参考图
    │    └─ 分镜故事板提示词 → GPT-Image-2 → 分镜参考图
    │
    ├─ Phase 3: 提示词生成（批量输出）
    │    └─ 每镜头Seedance提示词（英文描述+中文台词）
    │
    ├─ Phase 4: 视频生成（用户操作）
    │    └─ Seedance 2.0 全能参考
    │
    └─ Phase 5: 后期合成（用户操作）
         └─ 剪映/Premiere/达芬奇
```

### 触发词

| 触发词 | 功能 |
|--------|------|
| 一键成片 | 启动完整工作流 |
| 批量生成 | 批量输出所有提示词 |
| 生成设定板 | 生成所有设定板提示词 |
| 生成分镜 | 生成分镜脚本和故事板 |
| 生成Seedance提示词 | 生成所有镜头提示词 |

### 输出文件

```
项目/
├── 01_角色卡/
├── 02_设定板提示词/
├── 03_参考图/
├── 04_Seedance提示词/

```
