# 角色设定板提示词模板

## 模板一：左右分割式（你用的这个）

### 结构
```
左侧(1/2)：高质量角色主视觉半身肖像
右侧(1/2)：模块化角色信息系统
  ├─ 全身三视图（正面、侧面、背面）
  ├─ 角色比例统一
  ├─ 标准化角色身份资料
  └─ 面部五官细节特写
```

### 提示词模板
```
Character reference sheet with left-right split layout:

LEFT SIDE: High quality character main visual, 
[半身/胸像] portrait of [角色描述], 
[服装详细描述], [发型描述], [表情], 
[光影效果], cinematic, photorealistic, 
white clean background

RIGHT SIDE: Modular character information system, 
thin border分割的网格布局:
- Top: Full body three-view (front, side, back) 
  with unified proportions, 
  standardized character identity reference
- Bottom: Close-up facial feature details

Style: professional character design sheet,
unified color system, high information density layout,
realistic texture, cinematic lighting,
realistic skin texture, white clean background,
cinematic lighting, ultra high detail,
absolutely no text or labels
```

### 可修改变量

| 变量 | 说明 | 示例 |
|------|------|------|
| [角色描述] | 基础人设 | 19岁东方美女 |
| [服装详细描述] | 服装细节 | 广袖魏晋交领长裙绣金配色 |
| [发型描述] | 发型发色 | 黑长直发散落肩头 |
| [表情] | 面部表情 | 妩媚天真而不自知 |
| [光影效果] | 灯光设计 | 侧光，发丝轻飘 |

## 模板二：九宫格式（多角度）

### 结构
```
3×3 网格，9个不同角度：
正面 | 3/4正面 | 侧面
低角度 | 平视 | 高角度
背面 | 3/4背面 | 俯视
```

### 提示词模板
```
Character reference sheet in 3x3 grid layout, 
9 different perspectives of [角色]:
front view, 3/4 front view, side profile,
low angle, eye-level, high angle,
back view, 3/4 back view, overhead view,
[角色描述], [服装], [发型],
consistent lighting across all 9 frames,
uniform light warm gray background,
professional character design,
identical features, no text or labels
```
