# 道具设定板提示词模板

## 模板一：左右分割式（与角色设定板同构）

### 结构

```
左侧（1/2）：高质量道具主视觉视图
右侧（1/2）：模块化道具信息系统
  ├─ 上层：道具三视图（正面/侧面/背面）
  ├─ 中层：道具比例统一，标准化资料
  └─ 下层：道具各部位细节特写
```

### 提示词模板

```
道具描述: [道具名称], 真实质感。[材质描述]; [主题材质]。
[颜色撞色处理], [尺寸比例]，通过配饰规格强化整体感。
[细节特征]

画面风格:
色彩系统, 高信息密度排版, 真实质感
电影级光影, 真实[材质]纹理, 白色无干扰背景,
电影级灯光, 超高细节, 统一色彩系统

画面要求:
1. 左侧: 高质量道具主视觉视图
2. 右侧: 模块化道具信息系统, 采用大量细边框分割:
   1) 道具三视图(正面、侧面、背面)
   2) 其中道具比例统一
   3) 三视图标准化道具资料
   4) 道具各部位细节特写
```

### 英文版本（GPT-Image-2使用）

```

```

| 玻璃 | 透明、磨砂 | clear glass, frosted glass |
| 皮革 | 真皮、人造革 | genuine leather, synthetic leather |
| 木材 | 实木、胶合板 | solid wood, plywood |
| 塑料 | 亚克力、ABS | acrylic, ABS plastic |
| 碳纤维 | 碳纤维 | carbon fiber |
| 陶瓷 | 瓷器、陶器 | ceramic, porcelain |
