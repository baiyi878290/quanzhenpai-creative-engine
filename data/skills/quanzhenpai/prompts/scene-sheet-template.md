# 场景设定板提示词模板

## 模板一：实验室/办公场景（冷调）

### 结构

```
1. 景别指令
2. 风格指令（写实/电影感）
3. 画质指令（干净/锐利/无噪点）
4. 色调指令（低饱和/冷调/暖调）
5. 场景类型（开阔/封闭/室内/室外）
6. 核心元素（家具/设备/道具）
7. 背景元素（墙面/窗户/架子）
8. 顶部元素（灯具/吊顶）
9. 地面元素（地板/地毯）
10. 光影效果（亮调/柔光/硬光）
11. 特殊效果（柔焦/朦胧/通透）
12. 视角指令（俯拍/平视/仰拍）
```

### 示例：实验室场景

```
全景镜头，写实主义电影感，真人实拍。
画面干净锐利，无可见噪点。
低饱和度，冷调统一——主色调为冷白与灰蓝，
高光偏冷白，阴影偏青灰，
开阔明亮的大空间室内场景，
开阔空间冷白色主色调，淡蓝色低饱和度点缀，
(左侧白色长条实验操作台：1.1)，
台面上放置白蓝配色的浅色的光学显微镜，
和科学试剂，以及烧杯。
背景整面白色实验边台，开放式白色置物架，
透明玻璃实验器皿，左侧大面积落地窗，
白色集成吊顶嵌入长方形平板吸顶灯，
无缝白色洁净地坪，整体亮调，
柔和高光，轻微柔焦朦胧质感，
画面通透干净，有一个实验室的白色门，
俯拍视角拍摄
```

### 可替换变量

| 变量 | 说明 | 示例 |
|------|------|------|
| 景别 | 镜头范围 | 全景/中景/近景 |
| 色调 | 整体色彩倾向 | 冷白灰蓝/暖黄棕/中性灰 |
| 场景类型 | 空间特征 | 实验室/办公室/客厅/卧室 |
| 核心元素 | 主要物体 | 实验台/办公桌/沙发/床 |
| 背景元素 | 环境细节 | 置物架/窗户/墙面装饰 |
| 顶部元素 | 天花板/灯具 | 吊顶/吸顶灯/吊灯 |
| 地面元素 | 地板类型 | 洁净地坪/木地板/地毯 |
| 光影 | 灯光效果 | 亮调柔和/暗调戏剧/自然光 |
| 特殊效果 | 画面处理 | 柔焦/朦胧/通透/锐利 |
| 视角 | 拍摄角度 | 俯拍/平视/仰拍/斜角 |

## 使用指南

### Step 1: 确定场景类型
- 实验室/办公室 → 冷调模板
- 家居/咖啡厅 → 暖调模板
- 夜晚/悬疑 → 暗调模板
- 街道/公园 → 室外模板
- 科幻/奇幻 → 风格化模板

### Step 2: 填充变量
根据剧本场景描述，填充对应变量。

### Step 3: 调整细节
根据具体需求，增减元素描述。

### Step 4: 验证
检查清单：
- □ 色调是否统一？
- □ 光影方向是否一致？
- □ 元素是否完整？
- □ 视角是否正确？
