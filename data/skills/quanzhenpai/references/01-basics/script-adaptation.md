# 剧本改编指南

## 一、竖屏短剧改编核心原则

### 1.1 竖屏短剧特征
- **时长**：单集1-3分钟，总集数80-120集
- **节奏**：前3秒必有钩子，每30秒一个爽点/反转
- **画面**：9:16竖屏，以人物特写和近景为主
- **台词**：短句为主，每句不超过15字
- **情绪**：高密度情绪刺激，观众"爽感"优先

### 1.2 爆款公式
```
钩子（前3秒）→ 冲突建立（10秒）→ 小高潮（30秒）→ 悬念（结尾）
```

### 1.3 网络小说改编压缩法则

**保留什么：**
1. 核心人设（主角+反派+CP）
2. 主线冲突（最多保留2条）
3. 名场面（读者记忆点）
4. 爽点/虐点（情绪高潮）

**砍掉什么：**
1. 支线剧情（合并或删除）
2. 内心独白（转化为动作/台词）
3. 环境描写（压缩为1-2句）
4. 配角戏份（合并到主线）

**转化什么：**
1. 内心戏 → 表情特写+画外音
2. 回忆 → 闪回片段（3-5秒）
3. 旁白 → 角色自嘲式台词
4. 心理活动 → 微表情+肢体语言

## 二、改编流程

### 2.1 网络小说改编（6步）

```
Step 1: 提取核心
  - 主角人设（3句话）
  - 核心冲突（1句话）
  - 爽点清单（Top 10）
  - CP关系线

Step 2: 压缩结构
  - 原著章节 → 分集大纲
  - 每集1个核心事件
  - 每3集1个小高潮
  - 每10集1个大反转

Step 3: 分场设计
  - 每集3-5场戏
  - 每场1-2分钟
  - 场景不超过3个

Step 4: 台词改写
  - 长句变短句
  - 书面语变口语
  - 加入"金句"（可传播的台词）

Step 5: 钩子设计
  - 开头钩子：悬念/冲突/反转
  - 结尾钩子：留悬念给下一集
  - 中间钩子：每30秒刺激点

Step 6: 合规检查
  - 敏感词过滤
  - 暴力/色情尺度
  - 广电审核标准
```

### 2.2 漫画改编（4步）

```
Step 1: 分镜提取
  - 漫画分镜 → 影视分镜
  - 保留关键画面
  - 补充运动镜头

Step 2: 对话转化
  - 漫画气泡 → 台词
  - 加入潜台词
  - 补充动作描述

Step 3: 节奏调整
  - 漫画翻页感 → 影视节奏
  - 静态画面 → 动态镜头

Step 4: 风格统一
  - 画风 → 影视风格
  - 色彩 → 调色方案
```

### 2.3 真实事件改编（5步）

```
Step 1: 事件梳理
  - 时间线
  - 关键人物
  - 核心冲突
  - 转折点

Step 2: 戏剧化处理
  - 强化冲突
  - 合并人物
  - 压缩时间

Step 3: 人物塑造
  - 原型 → 角色
  - 加入人设标签
  - 设计人物弧光

Step 4: 结构重组
  - 线性 → 非线性（可选）
  - 加入悬念设计
  - 控制信息释放节奏

Step 5: 法律合规
  - 真实人物授权
  - 事件改编声明
  - 敏感内容处理
```

## 三、竖屏短剧剧本格式

```markdown
# 第X集：《集名》

## 场1：[场景] - [时间]
**画面**：[场景描述，1-2句]
**人物**：[出场人物]

[角色名]：（表情/动作）台词内容
[角色名]：台词内容

**[动作描述]**
角色做出某个动作

**[转场]**
切/叠/黑屏

---
## 场2：...

**本集结尾钩子**：[悬念描述]
```

## 四、爆款竖屏短剧类型模板

### 4.1 逆袭爽剧
```
核心：底层人物逆袭打脸
节奏：被欺负→反击→打脸→更大的敌人→再逆袭
爽点密度：每集至少1个打脸场景
```

### 4.2 甜宠剧
```
核心：双向奔赴的甜蜜恋爱
节奏：相遇→误会→接近→甜蜜→虐→更甜
爽点密度：每集至少1个心动瞬间
```

### 4.3 虐恋剧
```
核心：爱而不得的虐心故事
节奏：相爱→阻碍→分离→重逢→再分离→HE/BE
爽点密度：每集至少1个泪点
```

### 4.4 复仇剧
```
核心：被害者强势复仇
节奏：被害→隐忍→布局→反击→大仇得报
爽点密度：每集至少1个复仇进展
```

### 4.5 悬疑剧
```
核心：层层揭秘的悬疑故事
节奏：案件→线索→反转→真相→更大的秘密
爽点密度：每集至少1个新线索/反转
```
