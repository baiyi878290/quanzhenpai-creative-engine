## 二、STEP 5 质量审核详解

### 2.1 技术QC（FFprobe自动化）

| 检查项 | 标准 | 工具 | 严重等级 |
|--------|------|------|----------|
| 分辨率 | ≥1080p (1920×1080) | FFprobe | 🔴致命 |
| 帧率 | 24/25/30 fps | FFprobe | 🔴致命 |
| 音频响度 | -14 to -16 LUFS | FFmpeg loudnorm | 🟡严重 |
| 真峰值 | ≤ -1.0 dBTP | FFmpeg | 🟡严重 |
| 黑帧 | 0帧 | FFprobe scene检测 | 🔴致命 |
| 色彩空间 | Rec.709 | FFprobe | 🟢一般 |

### 2.2 画面QC

| 检查项 | 标准 | 工具 | 严重等级 |
|--------|------|------|----------|
| 画面清晰度 | 无模糊/失焦 | AI多模态分析 | 🟡严重 |
| 曝光准确性 | 无过曝/欠曝 | 直方图分析 | 🟡严重 |
| 色彩一致性 | 无色偏/跳色 | 色温分析 | 🟢一般 |
| 构图规范 | 符合景别要求 | AI视觉分析 | 🟢一般 |
| 字幕位置 | 安全区域内 | OCR检测 | 🟢一般 |

### 2.3 音频QC

| 检查项 | 标准 | 工具 | 严重等级 |
|--------|------|------|----------|
| 音画同步 | 误差≤40ms | 波形对齐 | 🔴致命 |
| 音量平衡 | 对白-12dBFS | 音频分析 | 🟡严重 |
| 背景噪音 | ≤-50dB | 噪声检测 | 🟢一般 |
| 音乐音量 | ≤-18dBFS | 音频分析 | 🟢一般 |

### 2.4 剪辑QC

| 检查项 | 标准 | 工具 | 严重等级 |
|--------|------|------|----------|
| 场景转换 | 无跳切/闪帧 | AI分析 | 🟡严重 |
| 节奏匹配 | 符合情绪曲线 | 节拍分析 | 🟢一般 |
| 转场自然 | 无突兀切换 | AI视觉 | 🟢一般 |
| 动作连贯 | 动作匹配 | AI视觉 | 🟡严重 |

### 2.5 美术QC

| 检查项 | 标准 | 工具 | 严重等级 |
|--------|------|------|----------|
| 服装时代 | 符合设定 | AI视觉 | 🟡严重 |
| 道具一致 | 无穿帮 | AI视觉 | 🟡严重 |
| 场景氛围 | 符合调性 | AI视觉 | 🟢一般 |
| 化妆造型 | 符合人设 | AI视觉 | 🟢一般 |

### 2.6 合规QC

| 检查项 | 标准 | 工具 | 严重等级 |
|--------|------|------|----------|
| 敏感词 | 无违禁内容 | 关键词库 | 🔴致命 |
| 暴力尺度 | 符合分级 | AI分析 | 🔴致命 |
| 色情内容 | 符合分级 | AI分析 | 🔴致命 |

## 三、质量评分体系

### 3.1 6维度评分

| 维度 | 权重 | 满分 | 评分标准 |
|------|------|------|----------|
| 技术QC | 20% | 100 | FFprobe自动化 |
| 画面QC | 25% | 100 | AI多模态分析 |
| 音频QC | 20% | 100 | 音频分析工具 |
| 剪辑QC | 15% | 100 | AI叙事分析 |
| 美术QC | 10% | 100 | AI视觉分析 |
| 合规QC | 10% | 100 | 关键词+AI |

### 3.2 综合评分

```
综合得分 = Σ(维度得分 × 权重)

评级标准：
- S级 (95-100)：直接发布
- A级 (85-94)：微调后发布
- B级 (70-84)：需修改
- C级 (60-69)：需重做部分
- D级 (<60)：需重做
```

## 四、退回机制

### 4.1 退回条件

| 条件 | 处理 |
|------|------|
| 🔴致命问题 ≥1个 | 立即退回 |
| 🟡严重问题 ≥3个 | 退回 |
| 🟢一般问题 ≥5个 | 建议修改 |

### 4.2 退回流程

```
退回 → 生成问题清单 → 标记位置+问题+建议 → STEP 4修改 → 重新提交STEP 5
```

### 4.3 超限处理

- 累计退回 >2轮 → 全崽硬裁
- 硬裁方式：对比两版差异，帧妞一句话定案

## 五、工具集成

### 5.1 FFprobe自动化脚本

```bash
# 技术QC检测
ffprobe -v quiet -print_format json -show_format -show_streams input.mp4

# 音频响度检测
ffmpeg -i input.mp4 -af loudnorm=print_format=json -f null -

# 场景检测（黑帧/场景切换）
ffprobe -v quiet -show_entries frame=pict_type -of csv=p=0 input.mp4
```

### 5.2 AI多模态分析

```python
# 画面QC调用示例
prompt = """

2. 曝光是否准确？
3. 构图是否符合[景别]要求？
4. 有无穿帮/不一致？
"""
```
