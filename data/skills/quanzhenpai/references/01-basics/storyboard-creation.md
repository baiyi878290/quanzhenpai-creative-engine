# 分镜脚本创作指南

## 一、分镜基础格式

### 1.1 标准分镜表

| 字段 | 说明 | 示例 |
|------|------|------|
| 镜号 | 唯一编号 | S001 |
| 景别 | 画面范围 | 特写/近景/中景/全景/远景 |
| 角度 | 拍摄角度 | 平视/俯拍/仰拍/斜角 |
| 运镜 | 镜头运动 | 推/拉/摇/移/跟/升/降/甩 |
| 画面描述 | 具体内容 | 女主角回头，眼含泪光 |
| 台词/音效 | 声音元素 | "我不后悔" + 背景音乐 |
| 时长 | 秒数 | 3s |
| 备注 | 特殊要求 | 慢动作/特效/转场 |

### 1.2 竖屏短剧分镜特点

**景别分布（推荐比例）：**
- 特写：30%（情绪表达）
- 近景：40%（人物互动）
- 中景：20%（场景交代）
- 全景/远景：10%（环境建立）

**运镜原则：**
- 竖屏以固定镜头为主（60%）
- 推拉镜头用于情绪强化（20%）
- 摇移镜头用于场景转换（15%）
- 运动镜头用于动作场面（5%）

## 二、双轨分镜系统

### 2.1 实拍分镜
为真人拍摄设计，包含：
- 具体的机位设置
- 灯光布置
- 演员调度
- 道具位置

### 2.2 AI分镜
为AI视频生成设计，包含：
- 提示词（Prompt）
- 参考图（Reference Image）
- 风格参数
- 一致性控制

### 2.3 双轨对照格式

```markdown
## 镜号 S001

### 实拍方案
- 景别：近景
- 机位：正面，高度1.2m
- 灯光：主光45°侧逆，辅光正面补光
- 演员：面对镜头，表情从微笑转为凝重
- 道具：桌上咖啡杯，冒热气

### AI方案
- 提示词：Close-up shot of a young woman, 
  expression changing from smile to serious, 
  warm indoor lighting, coffee cup on table 
  with steam rising, cinematic, shallow DOF
- 参考图：[角色参考表]
- 参数：9:16, 720p, 5s
```

## 三、景别详解

### 3.1 特写（Extreme Close-up）
**用途**：情绪表达、细节展示
**范围**：脸部/物体局部
**竖屏应用**：表情变化、眼神交流、嘴唇特写

**提示词模板**：
```
Extreme close-up of [主体], [表情/状态], 
[光影], [氛围], cinematic, shallow DOF
```

### 3.2 近景（Close-up）
**用途**：人物互动、对话场景
**范围**：胸部以上
**竖屏应用**：对话、内心戏、反应镜头

**提示词模板**：
```
Close-up shot of [人物], [动作], [表情], 
[环境暗示], [光影], cinematic
```

### 3.3 中景（Medium Shot）
**用途**：场景交代、动作展示
**范围**：膝盖以上
**竖屏应用**：走动、互动、场景切换

**提示词模板**：
```
Medium shot of [人物], [动作], [场景], 
[光影], [氛围], cinematic composition
```

### 3.4 全景（Wide Shot）
**用途**：环境建立、氛围营造
**范围**：全身+环境
**竖屏应用**：开场、转场、环境展示

**提示词模板**：
```
Wide shot of [场景], [人物位置], [环境细节], 
[光影], [氛围], cinematic establishing shot
```

## 四、运镜设计

### 4.1 推镜头（Push In）
**效果**：聚焦、强调、紧张感
**应用**：情绪递进、发现细节、悬疑营造

**提示词**：
```
Slow dolly push in on [主体], 
[情绪变化], [光影变化], cinematic
```

### 4.2 拉镜头（Pull Out）
**效果**：揭示、疏离、释然
**应用**：场景揭示、情绪释放、结尾

**提示词**：
```
Slow zoom out from [主体] revealing [全貌], 
[情绪], [光影], cinematic
```

### 4.3 摇镜头（Pan）
**效果**：巡视、跟随、引导
**应用**：场景介绍、人物跟随、视线引导

**提示词**：
```
Slow pan from [起点] to [终点], 
[场景], [光影], cinematic
```

### 4.4 移镜头（Tracking）
**效果**：运动感、沉浸感
**应用**：追逐、行走、动态场景

**提示词**：
```
Tracking shot following [人物] [动作], 
[场景], [速度], cinematic
```

### 4.5 升降镜头（Crane）
**效果**：史诗感、俯瞰、仰望
**应用**：开场、高潮、情绪升华

**提示词**：
```
Crane shot rising from [低位] to [高位], 
revealing [场景], [情绪], cinematic
```

## 五、竖屏分镜构图原则

### 5.1 人物构图
- **三分法**：人物置于竖向三分线
- **中心构图**：对称场景、强调情绪
- **留白**：上方留白给字幕空间

### 5.2 背景处理
- **虚化**：大光圈虚化背景
- **简洁**：避免杂乱背景
- **引导线**：利用线条引导视线

### 5.3 字幕空间
- 上方1/5：留白给标题/字幕
- 下方1/5：留白给操作栏
- 中间3/5：主体内容区

## 六、分镜脚本模板

```markdown
# 《剧名》第X集 分镜脚本

## 场1：[场景名称] - [时间]

### S001
- 景别：特写
- 运镜：固定
- 画面：女主角的眼睛，泪光闪烁
- 台词：（无）
- 音效：轻微啜泣声
- 时长：2s

### S002
- 景别：近景
- 运镜：缓慢推近
- 画面：女主角咬唇忍泪，手握紧拳头
- 台词：（内心OS）"我不能哭"
- 音效：心跳声渐强
- 时长：3s

### S003
- 景别：中景
- 运镜：固定
- 画面：女主角站起，转身背对男主角
- 台词：（对男主）"我们结束吧"
- 音效：（无）
- 时长：2s

---
## 场2：...
```

## 七、AI分镜一致性控制

### 7.1 角色一致性
- 使用角色参考表（多角度）
- 固定服装、发型、配饰
- 使用Seedance的"全能参考"功能

### 7.2 场景一致性
- 使用场景参考图
- 固定光影方向
- 保持色调统一

### 7.3 风格一致性
- 统一画面风格关键词
- 固定技术参数
- 使用风格参考图
