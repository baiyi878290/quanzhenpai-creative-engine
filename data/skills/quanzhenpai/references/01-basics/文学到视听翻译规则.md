# 不动如山 · 文学→视听翻译规则

## 核心原则
剧本不是小说的替代品。每一行字必须能直接或间接地转化为视听语言。做不到的——重写。

## 可拍 ✅（直接可用）

### 动作描写
```
他站起来。走出掩蔽部。头没碰到顶梁。
```
→ 摄影机：跟拍。演员走过低矮的出口，头低于顶梁。

### 物体特性
```
炮弹在弹壳上凝成了霜。他用手指摸——冰化了。水在指头上。
```
→ 摄影机：特写炮弹弹壳。霜。手指入画。霜化成水珠。

### 环境信息
```
风从弹孔里吹进来。哨声很细。
```
→ 现场录音：风声过弹孔的哨音。

### 数字
```
七发。七点六二毫米。够打死两个人。
```
→ 演员说。字幕不需要。观众不需要知道毫米，但需要知道"很少"。

## 不可拍 ❌（需翻译或删除）

### 心理状态直接陈述
```
他的眼睛红了。不是哭。是三天没合眼了。
```
→ 问题：前半句化妆可做（红眼睛），后半句需要观众自行推断。导演无法拍"是因为三天没合眼"。
→ 翻译：改为纯动作——
```
他揉了揉眼睛。手指从眼窝上移开——指节上的冻疮裂了。血珠渗出来。
```
→ 三个小时没睡的形象通过冻疮和揉眼的动作传递，不需要旁白解释。

### 情绪判断
```
不是怕。是失血。
```
→ 问题：这是作者告诉读者的，摄影机拍不到"不是怕"。
→ 翻译：删除判断，只描述可观察的动作——
```
他拆急救包。手指在发抖。
```
→ 观众自己判断是怕还是失血。

### 隐喻/比喻
```
像钉子在山上。
像抱着一个还在睡的孩子。
```
→ 这些是好的文学修辞，但不可直接拍。演员不能说"我抱着炮弹像抱着孩子"。
→ 处理：保留描写中的比喻，但确保比喻之前有纯动作版——
```
他把炮弹抱在怀里。站起来。
→ （导演/演员可以用这个比喻来理解动作意图，但不拍比喻本身）
```

### 时间跳跃
```
炮声从中午响到傍晚，没有停过。
```
→ 问题：观众感受不到"中午到傍晚"。
→ 翻译：用可见的变化替代——
```
油灯矮了一截。外面的光从白变成灰。
```

## 翻译三问（每场戏写完必问）

| # | 问题 | 如果答不出 |
|:--:|------|------|
| 1 | 摄影机放在哪？ | 重写空间关系 |
| 2 | 观众看到什么？ | 从内心独白改为外部动作 |
| 3 | 声音是什么？ | 添加环境音/对话/静默 |

## 剧本中的可拍范例（优秀）

- "他从怀里掏出枕套。白布。红线。四个字。" → 能拍
- "手指在'心'字上停着。不是摸。是停着。" → 前半句能拍，后半句删除 "不是摸" → "手指停在'心'字上。"
- "他叠好枕套。不是匆忙。是熟练——像做过一千遍的动作。" → 前半句能拍，"不是匆忙是熟练"删除

## 剧本中需翻译的句子（标记待处理）

以下句子在文学阅读中极好，但在拍摄时需要导演二次创作。建议在交付导演版时在旁边加【视听提示】：

| 原文（文学金句） | 视听提示 |
|------|------|
| "灰在光里不动——飘着飘着就停了。" | 【拍法：尘埃在光柱中的慢镜头。先浮动的微尘→停。】 |
| "安静得不真实。" | 【拍法：战后的战场，无声+耳鸣音效。】 |
| "他的背影和曹玉海的背影很像。不是身材像。是那个往前走的姿态。" | 【拍法：周二柱的主观视角。背影远去。不用叠化。】 |
| "山不动——是因为山下面都是山。" | 【拍法：周二柱的特写→拉远，运-20起飞的远景。群山。】 |
