# 构图法则库

## 一、12种专业构图法

### 1.1 三分法（Rule of Thirds）

```
┌───┬───┬───┐
│   │   │   │
├───┼───┼───┤
│   │ ● │   │  ← 主体置于交叉点
├───┼───┼───┤
│   │   │   │
└───┴───┴───┘
```

**原理：**
- 最经典的构图法则
- 将画面分为3×3网格
- 主体置于交叉点或线上
- 创造视觉平衡

**适用场景：**
- 通用，最安全的构图
- 人物对话
- 风景拍摄
- 产品展示

**AI提示词：**
```
subject placed at rule of thirds intersection,
balanced composition, thirds grid placement
```

**竖屏应用：**
```
人物构图：
- 眼睛在上三分线
- 身体在左或右三分线
- 避免人物居中（除非强调）
```

---

### 1.2 中心构图（Center Composition）

```
┌─────────────┐
│             │
│      ●      │  ← 主体居中
│             │
└─────────────┘
```

**原理：**
- 主体占据画面中心
- 创造稳定、庄重感
- 强调主体重要性

**适用场景：**
- 对称场景
- 仪式感
- 强调主体
- 霸气出场

**AI提示词：**
```
centered composition, symmetrical framing,
subject in center, balanced and stable
```

**竖屏应用：**
```
- 霸总出场：人物居中，气场强大
- 婚礼场景：新人居中，仪式感
- 对峙场景：双人居中，张力十足
```

---

### 1.3 引导线构图（Leading Lines）

```
    ╲     ╱
     ╲   ╱
      ╲ ╱
       ●      ← 线条引导视线到主体
      ╱ ╲
     ╱   ╲
    ╱     ╲
```

**原理：**
- 利用线条引导观众视线
- 线条汇聚到主体
- 创造纵深感

**适用场景：**
- 道路、走廊
- 河流、桥梁
- 建筑线条
- 光线引导

**AI提示词：**
```
leading lines guiding to subject,
converging lines, depth perspective,
road/corridor/path leading to subject
```

**竖屏应用：**
```
- 走廊场景：走廊线条引导到人物
- 街道场景：街道线条引导到主体
- 楼梯场景：楼梯线条引导到上方
```

---

### 1.4 框架构图（Frame within Frame）

```
┌─────────────┐
│ ┌─────────┐ │
│ │    ●    │ │  ← 门框/窗户框住主体
│ └─────────┘ │
└─────────────┘
```

**原理：**
- 利用前景元素框住主体
- 创造窥视感
- 增加层次感

**适用场景：**
- 门窗
- 拱门
- 树枝
- 任何框架元素

**AI提示词：**
```
subject framed by doorway/window/arch,
frame within frame composition,
natural frame, voyeuristic perspective
```

**竖屏应用：**
```
- 偷窥场景：门框框住被偷窥者
- 回忆场景：窗户框住回忆画面
- 秘密场景：树枝框住秘密场景
```

---

### 1.5 对角线构图（Diagonal Composition）

```
●────────────
 ╲
  ╲
   ╲
    ╲────────●  ← 主体沿对角线分布
```

**原理：**
- 创造动态感
- 增加张力
- 打破平衡

**适用场景：**
- 动作场景
- 冲突场景
- 追逐场景
- 需要张力的场景

**AI提示词：**
```
diagonal composition, dynamic framing,
subjects along diagonal line,
tension and movement
```

**竖屏应用：**
```
- 打斗场景：两人沿对角线对峙
- 追逐场景：追逐方向沿对角线
- 冲突场景：冲突双方沿对角线
```

---

### 1.6 前景层次构图（Foreground Layering）

```
┌─────────────┐
│   远景      │
│   中景  ●   │  ← 前景-中景-远景层次
│   前景      │
└─────────────┘
```

**原理：**
- 创造空间深度
- 增加层次感
- 引导视线

**适用场景：**
- 需要空间感
- 环境交代
- 深度感需求

**AI提示词：**
```
foreground elements, depth layers,
foreground middle ground background,
layered composition, spatial depth
```

**竖屏应用：**
```
- 咖啡厅场景：前景杯子，中景人物，远景窗
- 街道场景：前景栏杆，中景人物，远景建筑
- 卧室场景：前景床单，中景人物，远景窗户
```

---

### 1.7 负空间构图（Negative Space）

```
┌─────────────┐
│             │
│             │
│          ●  │  ← 大量留白，主体偏小
│             │
└─────────────┘
```

**原理：**
- 大量留白
- 突出主体渺小
- 创造孤独感

**适用场景：**
- 孤独场景
- 渺小感
- 意境表达
- 极简风格

**AI提示词：**
```
negative space, minimal composition,
subject small in frame, vast empty space,
lonely, isolated, minimalist
```

**竖屏应用：**
```
- 孤独场景：人物在画面底部，上方大量留白
- 思念场景：人物望向留白方向
- 失落场景：人物被空间包围
```

---

### 1.8 填满构图（Fill the Frame）

```
┌─────────────┐
│  ┌───────┐  │
│  │  ●●●  │  │  ← 主体填满画面
│  │  ●●●  │  │
│  └───────┘  │
└─────────────┘
```

**原理：**
- 主体填满整个画面
- 创造亲密感
- 强调细节

**适用场景：**
- 特写镜头
- 细节展示
- 压迫感
- 亲密场景

**AI提示词：**
```
tight framing, fill the frame,
subject filling entire frame,
close and intimate, claustrophobic
```

**竖屏应用：**
```
- 情绪特写：脸部填满画面
- 压迫场景：被压迫者脸部填满
- 亲密场景：两人脸部填满画面
```

---

### 1.9 三角构图（Triangle Composition）

```
      ●
     ╱ ╲
    ╱   ╲
   ●─────●  ← 三点构成三角
```

**原理：**
- 三点构成稳定结构
- 创造视觉平衡
- 适合多人构图

**适用场景：**
- 三人场景
- 家庭场景
- 稳定关系

**AI提示词：**
```
triangle composition, three subjects,
stable triangular arrangement,
balanced group composition
```

**竖屏应用：**
```
- 家庭场景：父母+孩子构成三角
- 三角恋：三人构成三角
- 对话场景：两人+道具构成三角
```

---

### 1.10 对称构图（Symmetry）

```
┌─────────────┐
│  ●       ●  │
│    ●   ●    │  ← 完全对称
│      ●      │
└─────────────┘
```

**原理：**
- 完全对称
- 创造秩序感
- 庄严、完美

**适用场景：**
- 建筑场景
- 仪式场景
- 梦境场景
- 完美场景

**AI提示词：**
```
perfect symmetry, mirrored composition,
symmetrical framing, balanced and orderly,
architectural symmetry
```

**竖屏应用：**
```
- 婚礼场景：新人对称站立
- 建筑场景：对称建筑线条
- 梦境场景：超现实对称
```

---

### 1.11 黄金螺旋（Golden Spiral）

```
┌─────────────┐
│         ╭──╮│
│      ╭──╯  ││
│   ╭──╯  ●  ││  ← 主体在螺旋终点
│   │        ││
└───┴────────┴┘
```

**原理：**
- 基于斐波那契数列
- 最自然的构图
- 视觉流动感

**适用场景：**
- 自然场景
- 优雅场景
- 流动感需求

**AI提示词：**
```
golden spiral composition, fibonacci spiral,
natural flow, elegant framing,
spiral leading to subject
```

---

### 1.12 透视构图（Perspective）

```
      ●
     ╱│╲
    ╱ │ ╲
   ╱  │  ╲
  ───────────  ← 透视汇聚
```

**原理：**
- 透视线条汇聚
- 创造纵深感
- 强烈的空间感

**适用场景：**
- 道路、走廊
- 建筑、街道
- 需要纵深感

**AI提示词：**
```
perspective lines, vanishing point,
converging perspective, strong depth,
one-point perspective
```

## 三、构图与情绪

### 3.1 构图情绪对照

| 构图 | 情绪 | 适用场景 |
|------|------|----------|
| 中心构图 | 稳定、重要 | 庄严、强调 |
| 三分法 | 平衡、自然 | 通用 |
| 对称构图 | 秩序、完美 | 建筑、梦境 |
| 不对称构图 | 动态、不安 | 冲突、悬疑 |
| 框架构图 | 窥视、秘密 | 偷窥、秘密 |
| 负空间 | 孤独、渺小 | 孤独、思念 |
| 填满构图 | 压迫、亲密 | 情绪爆发 |
| 对角线 | 张力、冲突 | 打斗、追逐 |

### 3.2 构图与角色关系

| 关系 | 推荐构图 | 说明 |
|------|----------|------|
| 亲密 | 填满构图、双人特写 | 拉近距离 |
| 对立 | 对角线、分割构图 | 创造张力 |
| 疏离 | 负空间、远景 | 拉开距离 |
| 权力 | 仰拍+中心、俯拍+边缘 | 体现强弱 |
| 平等 | 三分法、对称 | 体现平衡 |

## 四、AI构图提示词库

### 4.1 基础构图词

```
三分法：rule of thirds, thirds grid
中心构图：centered, central composition
对称构图：symmetrical, mirrored
引导线：leading lines, converging lines
框架构图：framed, frame within frame
负空间：negative space, minimal
填满构图：tight framing, fill the frame
对角线：diagonal, dynamic
三角构图：triangular, three-point
透视：perspective, vanishing point
```

### 4.2 竖屏构图词

```
竖屏：vertical composition, 9:16, portrait
人物居中：subject centered vertically
人物偏左：subject on left third
人物偏右：subject on right third
眼睛在上：eyes at upper third
头顶留白：headroom, space above head
底部留白：footroom, space below
```

### 4.3 层次构图词

```
前景元素：foreground element, foreground framing
中景主体：middle ground subject
背景环境：background environment
深度层次：depth layers, layered composition
空间感：spatial depth, sense of space
```

### 4.4 完整构图提示词

```
# 三分法+浅景深
Subject placed at rule of thirds intersection,
shallow depth of field, background blurred,
balanced composition, cinematic, 9:16

# 框架构图+引导线
Subject framed by doorway, leading lines from frame to subject,
depth perspective, cinematic lighting, 9:16

# 负空间+情绪
Subject small in lower right corner,
vast empty space above, lonely atmosphere,
minimalist composition, cinematic, 9:16

# 对角线+动态
Two subjects along diagonal line,
dynamic composition, tension and movement,
dramatic lighting, cinematic, 9:16
```

## 六、构图禁忌

### 6.1 常见错误

```
❌ 切头切脚：人物头顶或脚部被切掉
❌ 头顶长树：背景物体从头顶长出
❌ 人物居中无意义：没有强调需求却居中
❌ 视线无留白：人物视线方向没有空间
❌ 地平线倾斜：水平线不水平
❌ 背景杂乱：背景元素干扰主体
❌ 主体太小：人物在画面中太小
❌ 主体太大：人物贴边，没有呼吸空间
```

### 6.2 竖屏特有禁忌

```
❌ 人物太靠上：头顶被切或贴顶
❌ 人物太靠下：进入字幕区域
❌ 忽略安全区：重要内容在安全区外
❌ 横屏思维：用横屏构图做竖屏
❌ 信息量过大：竖屏空间有限，元素过多
```

## 七、AI构图优化技巧

### 7.1 构图强化词

```
如果AI构图不理想，添加强化词：

加强三分法：
rule of thirds composition, subject at intersection point

加强对称：
perfect symmetry, perfectly mirrored

加强引导线：
strong leading lines, converging perspective

加强框架：
natural frame around subject, frame within frame
```

### 7.2 构图纠正词

```
如果AI构图有错误，添加纠正词：

修正切头：
proper headroom, space above head

修正视线：
space in direction of gaze, look space

修正位置：
subject properly positioned, balanced placement

修正层次：
foreground middle ground background, layered depth
```

### 7.3 构图组合提示词

```
# 完整构图描述
[构图类型], [主体位置], [视角], [层次], [氛围]

示例：
Rule of thirds composition, subject at right intersection,
eye level angle, foreground elements framing,
depth layers, cinematic atmosphere, 9:16
```
