# 情绪层次理论

## 一、情绪7层递进模型

### 1.1 核心原理

```
真实的情绪变化不是「瞬间切换」，而是「层层递进」

❌ 错误：听到坏消息 → 立刻大哭
✅ 正确：听到坏消息 → 停顿 → 理解 → 难以置信 → 接受 → 情绪涌上 → 表达 → 释放
```

### 1.2 七层模型

| 层次 | 名称 | 时间 | 面部 | 眼神 | 肢体 | 呼吸 |
|------|------|------|------|------|------|------|
| 1 | 听到/看到 | 0.2-0.3s | 凝固 | 凝固 | 静止 | 屏息 |
| 2 | 理解 | 0.3-0.5s | 微变 | 闪动 | 微颤 | 浅呼吸 |
| 3 | 难以置信 | 0.5-1s | 疑惑 | 眯眼/睁大 | 摇头/后退 | 浅快 |
| 4 | 接受 | 0.5-1s | 下垂 | 失焦 | 塌肩 | 深呼吸 |
| 5 | 情绪涌上 | 1-2s | 颤抖 | 泛红/含泪 | 握拳/捂嘴 | 颤抖 |
| 6 | 表达 | 2-3s | 哭泣/愤怒 | 泪流/瞪视 | 发抖/攻击 | 急促 |
| 7 | 释放 | 3-5s | 爆发/崩溃 | 失神/紧闭 | 瘫倒/摔砸 | 喘息 |

## 二、基础情绪层次详解

### 2.1 悲伤（听到噩耗）

```
S001 (0.3s): 特写，眼睛凝固，嘴巴微张
            "听到消息的瞬间"
            
S002 (0.5s): 特写，眉毛微皱，眼神闪动
            "理解消息的含义"
            
S003 (1s): 近景，摇头，嘴巴颤抖
          "不愿相信"
          
S004 (0.5s): 近景，眼神失焦，肩膀下垂
            "不得不接受"
            
S005 (1.5s): 特写，眼眶泛红，嘴唇颤抖
            "情绪开始涌上"
            
S006 (2s): 特写，泪水涌出，咬住下唇
          "试图控制"
          
S007 (3s): 近景，泪流满面，捂住脸
          "完全释放"
```

```

S007: Close-up, tears streaming, hands covering face,

```

### 2.2 愤怒（被背叛）

```
S001 (0.3s): 特写，眼睛凝固
            "听到/看到背叛"
            
S002 (0.3s): 特写，眼神锐利
            "理解发生了什么"
            
S003 (0.5s): 特写，眉毛紧皱
          "难以置信"
          
S004 (0.5s): 特写，咬紧牙关
            "愤怒开始"
            
S005 (1s): 近景，鼻翼张开，眼神凶狠
          "愤怒涌上"
          
S006 (1s): 近景，握紧拳头
          "试图控制"
          
S007 (2s): 近景/中景，爆发（拍桌/摔东西/攻击）
          "完全释放"
```

**AI提示词：**
```

S002: Extreme close-up, eyes sharpening, realization,
      understanding, dramatic lighting, cinematic, 9:16

S003: Extreme close-up, eyebrows deeply furrowed,
      disbelief, dramatic lighting, cinematic, 9:16

S004: Extreme close-up, jaw clenching, teeth grinding,
      anger building, dramatic lighting, cinematic, 9:16

S005: Close-up, nostrils flaring, fierce eyes,
      rage rising, dramatic lighting, cinematic, 9:16

S006: Close-up, fists clenching, knuckles white,
      trying to control, dramatic lighting, cinematic, 9:16

S007: Medium shot, explosive action, slamming table,
      complete release, dramatic lighting, cinematic, 9:16
```

### 2.3 喜悦（收到好消息）

```
S001 (0.2s): 特写，眼睛睁大
            "听到好消息"
            
S002 (0.3s): 特写，眉毛上扬
            "理解好消息"
            
S003 (0.5s): 特写，嘴巴微张
          "难以置信"
          
S004 (0.5s): 特写，嘴角上扬
            "开始接受"
            
S005 (1s): 近景，笑容扩大，眼睛发亮
          "喜悦涌上"
          
S006 (1s): 近景，笑容灿烂，可能含泪
          "喜悦表达"
          
S007 (2s): 中景，欢呼/拥抱/跳跃
          "完全释放"
```

### 2.4 恐惧（遇到危险）

```
S001 (0.2s): 特写，瞳孔放大
            "看到危险"
            
S002 (0.2s): 特写，眼睛睁大
            "理解危险"
            
S003 (0.3s): 特写，身体僵硬
          "本能反应"
          
S004 (0.5s): 特写，脸色发白
            "恐惧开始"
            
S005 (1s): 近景，身体颤抖，后退
          "恐惧涌上"
          
S006 (1s): 近景，嘴巴张开想叫但叫不出
          "试图求救"
          
S007 (2s): 中景，尖叫/逃跑/瘫倒
          "完全释放"
```

### 2.5 惊讶（意外发现）

```
S001 (0.2s): 特写，眼睛凝固
            "看到意外事物"
            
S002 (0.2s): 特写，眉毛上扬
            "理解意外"
            
S003 (0.3s): 特写，眼睛睁大
          "难以置信"
          
S004 (0.5s): 特写，嘴巴微张
            "接受惊讶"
            
S005 (1s): 近景，根据内容转为其他情绪
          "情绪转化"
```

## 三、复合情绪层次

### 3.1 心碎（被拒绝）

```
层次1：震惊（0.5s）
- 眼睛凝固
- 身体僵硬

层次2：难以置信（1s）
- 眉毛微皱
- 轻微摇头

层次3：疼痛（1s）
- 嘴唇颤抖
- 手捂胸口

层次4：悲伤（2s）
- 眼眶泛红
- 泪水涌出

层次5：释放（3s）
- 泪流满面
- 可能蹲下/扶墙
```

**AI提示词：**
```
Close-up, expression transitioning from shock to disbelief to pain to sadness,
eyes frozen then tearing up, lips trembling, hand moving to chest,
heartbreak moment, soft backlight, cinematic, 9:16
```

### 3.2 嫉妒（看到喜欢的人和别人在一起）

```
层次1：注意到（0.3s）
- 眼神被吸引

层次2：确认（0.5s）
- 眼睛微眯
- 表情变冷

层次3：酸涩（1s）
- 咬住下唇
- 眼神闪烁

层次4：愤怒（1s）
- 眼神锐利
- 握紧拳头

层次5：压抑（2s）
- 转移视线
- 深呼吸
- 表情恢复（但眼神出卖）
```

### 3.3 心动（第一次被吸引）

```
层次1：注意到（0.3s）
- 眼神被吸引

层次2：停留（0.5s）
- 眼睛微微睁大
- 呼吸微顿

层次3：欣赏（1s）
- 眼神柔和
- 嘴角微扬

层次4：害羞（1s）
- 眼神躲闪
- 脸颊泛红

层次5：回味（2s）
- 忍不住再看
- 嘴角上扬
```

**AI提示词：**
```
Close-up, expression subtly changing from noticing to admiration to shyness,
eyes softening, slight smile appearing, cheeks flushing,
first attraction moment, soft golden light, cinematic, 9:16
```

### 3.4 隐忍（被冤枉但不能解释）

```
层次1：听到（0.3s）
- 眼睛微睁

层次2：愤怒（0.5s）
- 眉毛紧皱
- 眼神锐利

层次3：想反驳（0.5s）
- 嘴唇微张
- 又闭上

层次4：压抑（1s）
- 咬紧牙关
- 深呼吸

层次5：忍下（2s）
- 眼神暗淡
- 肩膀微垂
- 拳头握紧又松开
```

## 四、情绪转换层次

### 4.1 悲伤→愤怒

```
S001: 悲伤 - 泪流满面
S002: 回忆 - 想起原因
S003: 酸涩 - 嘴唇紧闭
S004: 愤怒 - 眼神变锐利
S005: 爆发 - 擦掉眼泪，眼神凶狠
```

### 4.2 愤怒→悲伤

```
S001: 愤怒 - 眼神凶狠
S002: 冲突 - 争执/打斗
S003: 受伤 - 身体或心理
S004: 软化 - 眼神变柔
S005: 悲伤 - 泪水涌出
```

### 4.3 恐惧→勇气

```
S001: 恐惧 - 身体颤抖
S002: 挣扎 - 内心斗争
S003: 决心 - 眼神变坚定
S004: 勇气 - 身体站直
S005: 行动 - 迈步向前
```

### 4.4 憎恨→释然

```
S001: 憎恨 - 眼神凶狠
S002: 对峙 - 面对对方
S003: 理解 - 了解真相
S004: 软化 - 眼神变化
S005: 释然 - 长叹，眼神平静
```

## 九、情绪层次检查清单

### 9.1 单镜头检查

```
□ 表情是否有变化？（即使微小）
□ 眼神是否有情绪？
□ 是否有微表情？
□ 呼吸是否自然？
□ 肢体是否配合？
```

### 9.2 镜头序列检查

```
□ 情绪是否有递进？
□ 转换是否自然？
□ 节奏是否合理？
□ 高潮是否突出？
□ 释放是否充分？
```

### 9.3 常见错误

```
❌ 情绪跳跃：直接从平静到崩溃，缺少中间层次
❌ 表情单一：整场戏只有一个表情
❌ 眼神空洞：没有情绪的眼神
❌ 肢体僵硬：身体没有配合情绪
❌ 节奏过快：情绪变化太快，观众跟不上
```

## 十、AI情绪生成技巧

### 10.1 情绪层次提示词

```
# 基础版
sad expression, tears in eyes

# 进阶版
expression transitioning from shock to sadness,
eyes welling up with tears, lips trembling,
emotional breakdown moment

# 专业版
subtle emotional progression: initial freeze,
eyes losing focus, understanding dawning,
lips starting to tremble, eyes glistening,
tears forming but held back, finally releasing
```

### 10.2 情绪强度控制

```
低强度：subtle, slight, hint of
中强度：clear, noticeable, evident
高强度：intense, overwhelming, powerful
极端：extreme, explosive, devastating
```

### 10.3 情绪组合提示词

```
悲伤+愤怒：sadness mixed with anger, tearful but fierce
开心+泪水：happy tears, crying with joy
恐惧+勇气：scared but determined, trembling with resolve
愤怒+悲伤：angry tears, furious yet heartbroken
```
