## 二、经典光效

### 2.1 伦勃朗光（Rembrandt Lighting）

```
特点：
- 侧光45度
- 面部形成三角形亮区
- 另一侧阴影

效果：
- 深度、神秘
- 人物立体感强

适用：
- 人物塑造
- 对话场景
- 悬疑氛围

AI提示词：
Rembrandt lighting, 45-degree side light,
triangle of light on cheek, dramatic shadows,
cinematic portrait, 9:16
```

### 2.2 蝴蝶光（Butterfly Lighting）

```
特点：
- 正面上方45度
- 鼻下形成蝴蝶形阴影
- 面部均匀照亮

效果：
- 优雅、美化
- 皮肤质感好

适用：
- 女性特写
- 时尚拍摄
- 美妆展示

AI提示词：
butterfly lighting, overhead front light,
even facial illumination, flattering,
beauty lighting, cinematic, 9:16
```

### 2.3 分割光（Split Lighting）

```
特点：
- 侧面90度
- 面部一半亮一半暗
- 强烈明暗对比

效果：
- 双重性、冲突
- 神秘、危险

适用：
- 内心挣扎
- 反派人物
- 悬疑场景

AI提示词：
split lighting, half face lit half in shadow,
dramatic contrast, mysterious,
dark and light duality, cinematic, 9:16
```

### 2.4 环形光（Loop Lighting）

```
特点：
- 正面30-45度
- 鼻下形成小环形阴影
- 柔和自然

效果：
- 自然、柔和
- 日常感

适用：
- 对话场景
- 日常场景
- 采访场景

AI提示词：
loop lighting, soft front light,
natural shadow pattern, flattering,
everyday lighting, cinematic, 9:16
```

### 2.5 轮廓光（Rim Lighting）

```
特点：
- 背后打光
- 人物边缘发光
- 与背景分离

效果：
- 分离、神圣
- 唯美、梦幻

适用：
- 人物突出
- 唯美场景
- 神圣时刻

AI提示词：
rim lighting, backlit, edge light,
subject glowing, separated from background,
ethereal, cinematic, 9:16
```

## 三、情绪光影方案

### 3.1 甜蜜光影

```
光质：柔光
色温：3200K-4000K（暖）
方向：正面或45度侧光
特点：高光柔和、阴影少、皮肤质感好

道具：纱布柔光、暖色滤纸

AI提示词：
soft warm lighting, golden hour glow,
gentle shadows, romantic atmosphere,
flattering light, cinematic, 9:16
```

### 3.2 悲伤光影

```
光质：柔光+硬光混合
色温：4000K-5000K（中性偏冷）
方向：顶光或侧逆光
特点：面部有阴影、眼神光弱、整体偏暗

道具：黑旗遮挡

AI提示词：
moody lighting, cool tones, subtle shadows,
melancholic atmosphere, dim and somber,
emotional lighting, cinematic, 9:16
```

### 3.3 愤怒光影

```
光质：硬光
色温：5000K-6500K（冷）
方向：侧光或底光
特点：明暗对比强、阴影深、面部半明半暗

道具：遮光板、格栅

AI提示词：
harsh dramatic lighting, strong contrast,
deep shadows, intense atmosphere,
angry mood, cinematic, 9:16
```

### 3.4 恐惧光影

```
光质：硬光
色温：4000K-5000K（冷）
方向：底光或侧光
特点：阴影扭曲、面部变形、不安感

道具：低位灯光

AI提示词：
eerie lighting, low angle light,
distorted shadows, horror atmosphere,
fearful mood, cinematic, 9:16
```

### 3.5 浪漫光影

```
光质：极柔光
色温：3000K-3500K（暖）
方向：逆光或侧逆光
特点：光晕、镜头光斑、梦幻感

道具：烟雾、纱布

AI提示词：
dreamy backlight, soft glow, lens flare,
romantic haze, ethereal atmosphere,
love scene lighting, cinematic, 9:16
```

### 3.6 悬疑光影

```
光质：硬光
色温：4500K-5500K（中性偏冷）
方向：侧光或顶光
特点：阴影深、对比强、局部照明

道具：束光筒、格栅

AI提示词：
noir lighting, dramatic shadows,
partial illumination, mysterious atmosphere,
suspenseful mood, cinematic, 9:16
```

## 四、场景光影方案

### 4.1 办公室

```
主光：窗户自然光或模拟窗户光
辅光：天花板荧光灯效果
色温：5000K-5500K
特点：明亮、均匀、专业感

AI提示词：
modern office lighting, fluorescent overhead,
window light from side, professional atmosphere,
even illumination, cinematic, 9:16
```

### 4.2 卧室

```
主光：床头灯或台灯
辅光：窗外月光（蓝调）
色温：2700K-3200K
特点：温馨、私密、柔和

AI提示词：
bedroom lighting, warm bedside lamp,
soft intimate glow, cozy atmosphere,
romantic lighting, cinematic, 9:16
```

### 4.3 酒吧

```
主光：霓虹灯/射灯
辅光：烟雾+光束
色温：混合（RGB）
特点：高饱和、明暗交替、迷幻

AI提示词：
neon bar lighting, colorful lights,
smoke and light beams, nightlife atmosphere,
vibrant and moody, cinematic, 9:16
```

### 4.4 医院

```
主光：顶灯（荧光灯）
辅光：无影灯效果
色温：5500K-6500K
特点：冷白、均匀、无阴影、刺眼

AI提示词：
hospital lighting, cold fluorescent overhead,
clinical white light, sterile atmosphere,
harsh and clean, cinematic, 9:16
```

### 4.5 夜景

```
主光：路灯/车灯
辅光：霓虹灯/窗户光
色温：混合（暖+冷）
特点：高对比、局部照明、氛围感

AI提示词：
night scene lighting, street lamps,
neon reflections, mixed color temperature,
urban night atmosphere, cinematic, 9:16
```

## 五、AI光影提示词库

### 5.1 光质词

```
柔光：soft light, diffused light, gentle light
硬光：hard light, harsh light, direct light
自然光：natural light, ambient light
人工光：artificial light, studio light
```

### 5.2 光向词

```
正面光：front light, flat light
侧面光：side light, lateral light
背面光：backlight, rim light
顶光：top light, overhead light
底光：under light, low angle light
45度光：45-degree light, Rembrandt light
```

### 5.3 光色词

```
暖光：warm light, golden light, amber glow
冷光：cool light, blue light, cold tones
中性光：neutral light, daylight balanced
混合光：mixed lighting, color contrast
```

### 5.4 光效词

```
光晕：glow, haze, soft glow
光斑：lens flare, light spots
体积光：volumetric light, god rays
轮廓光：rim light, edge light, halo
剪影：silhouette, backlit silhouette
丁达尔效应：Tyndall effect, light beams
```
