# 甜宠剧方法论

## 一、甜宠剧核心公式

### 1.1 爽点公式

```
甜蜜 = 误会 + 接近 + 心动 + 官宣 + 撒糖

每集必备：
- 1个误会（制造冲突）
- 1个接近（增加互动）
- 1个心动（甜蜜瞬间）
- 1个悬念（留住观众）
```

### 1.2 情绪曲线

```
情绪
  ↑
高│        ╱╲      ╱╲
  │       ╱  ╲    ╱  ╲

低│╱                      ╲
  └──────────────────────────→ 时间
   铺垫  甜蜜  误会  和好  更甜
```

## 二、甜宠剧视听规范

### 2.1 画面规范

```
色调：暖橙+粉色（色温6500K）
饱和度：中高（+15%）
对比度：中（避免过强）
景别：近景为主（60%），特写（30%）
运镜：缓慢、柔和、避免剧烈运动
构图：双人构图为主，保持亲密距离
景深：浅景深，背景虚化
光线：柔光、暖光、避免硬光
```

### 2.2 声音规范

```
配乐：钢琴/吉他抒情曲，轻快节奏
音效：减少尖锐音效，增加柔和音效
对白：语速稍慢，语调温柔
环境音：降低音量，突出对白
心跳声：心动时刻加入心跳音效
```

### 2.3 剪辑规范

```
节奏：中慢节奏，给观众「嗑糖」时间
镜头时长：3-5秒，避免过快切换
转场：淡入淡出、溶解为主
反应镜头：多用反应镜头（微笑、心动）
慢动作：心动瞬间使用慢动作
```

## 四、甜宠禁忌

### 4.1 画面禁忌

```
❌ 画面太暗：失去甜蜜感
❌ 画面太冷：失去温暖感
❌ 景别太远：失去亲密感
❌ 运镜太快：失去浪漫感
❌ 光影太硬：失去柔和感
❌ 构图太满：失去呼吸感
```

### 4.2 声音禁忌

```
❌ 配乐太紧张：破坏甜蜜氛围
❌ 音效太尖锐：破坏柔和感
❌ 对白太快：失去温柔感
❌ 环境音太吵：干扰情感表达
```

### 4.3 剪辑禁忌

```
❌ 节奏太快：观众来不及「嗑糖」
❌ 转场太硬：破坏浪漫氛围
❌ 反应镜头少：错过甜蜜瞬间
❌ 没有慢动作：错过心动时刻
```

### 4.4 表演禁忌

```
❌ 表情太严肃：失去甜蜜感
❌ 眼神太空洞：失去情感
❌ 肢体太僵硬：失去亲密感
❌ 反应太迟钝：错过心动瞬间
```

## 六、甜宠剧节奏模板

### 6.1 单集节奏（3分钟）

```
0:00-0:05  钩子：甜蜜预告/误会开始
0:05-0:30  铺垫：日常互动
0:30-1:00  发展：误会加深/接近机会
1:00-1:30  高潮：甜蜜瞬间/心动时刻
1:30-2:00  转折：误会爆发/冲突
2:00-2:30  和好：解释清楚/和解
2:30-3:00  结尾：更甜蜜/下集预告
```

### 6.2 情绪节奏

```
甜蜜：60%
误会：20%
心动：15%
悬念：5%
```
