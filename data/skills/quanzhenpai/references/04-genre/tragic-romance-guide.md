# 虐恋剧方法论

## 一、虐恋剧核心公式

### 1.1 爽点公式

```
虐心 = 相爱 + 阻碍 + 分离 + 重逢 + 再分离 + HE/BE

每集必备：
- 1个甜蜜瞬间（给希望）
- 1个阻碍出现（制造冲突）
- 1个误会加深（增加痛苦）
- 1个悬念（留住观众）
```

## 二、虐恋剧视听规范

### 2.1 画面规范

```
色调：冷蓝+灰（色温4500K）
饱和度：低-中（-20%）
对比度：中高
景别：特写为主（50%），近景（35%）
运镜：缓慢、沉重
构图：负空间、边缘构图
景深：浅景深，背景虚化
光线：冷光、阴影、逆光
```

### 2.2 声音规范

```
配乐：大提琴、钢琴悲伤曲
音效：雨声、风声、钟声
对白：低沉、颤抖、含泪
环境音：安静、空旷
```

### 2.3 剪辑规范

```
节奏：慢节奏，给观众「心痛」时间
镜头时长：4-6秒，避免过快切换
转场：溶解、闪白、闪黑
慢动作：分离时刻、泪水落下
```

### 3.1 含泪对视

```
镜头设计：

S002: 特写，B眼眶泛红
S003: 特写，两人对视
S004: 特写，A泪水落下
S005: 特写，B泪水落下

提示词：

```

### 3.2 背影离去

```
镜头设计：
S001: 中景，A转身
S002: 近景，A背影
S003: 特写，B看着背影
S004: 全景，A走远
S005: 特写，B泪水落下

提示词：
Medium shot of person walking away,
back view, leaving behind,
lonely atmosphere, cinematic, 9:16
```

### 3.3 雨中分离

```
镜头设计：
S001: 全景，两人在雨中
S002: 近景，A说分手
S003: 特写，B震惊
S004: 特写，雨水混着泪水
S005: 全景，A离开

提示词：
Two people in rain, separating,
raindrops mixing with tears,
heartbreak in rain, cinematic, 9:16
```

### 3.4 回忆闪回

```
镜头设计：
S001: 闪白过渡
S002: 过去的甜蜜（柔焦，暖色调）
S003: 过去的拥抱（慢动作）
S004: 闪白回到现在
S005: 现在的孤独（冷色调）

提示词：
Dreamy flashback, soft focus, warm tones,
nostalgic memory, then cut to cold present,
contrast of past and present, cinematic, 9:16
```

### 3.5 独自哭泣

```
镜头设计：
S001: 近景，A独自一人
S002: 特写，A咬住嘴唇
S003: 特写，泪水涌出
S004: 特写，A捂住脸
S005: 全景，A蜷缩在角落

提示词：
Close-up of person crying alone,
tears streaming, covering face,
solitary grief, cinematic, 9:16
```

### 3.6 重逢错过

```
镜头设计：
S001: 中景，A在人群中
S002: 中景，B在人群中
S003: 特写，A似乎看到B
S004: 特写，B似乎看到A
S005: 全景，人群走过，错过

提示词：
Two people in crowd, almost seeing each other,
near miss, passing by,
missed connection, cinematic, 9:16
```

## 四、虐恋禁忌

### 4.1 画面禁忌

```
❌ 画面太暖：失去虐心感
❌ 景别太远：失去情绪冲击
❌ 运镜太快：失去沉重感
❌ 光影太亮：失去压抑感
```

### 4.2 声音禁忌

```
❌ 配乐太欢快：破坏虐心氛围
❌ 对白太平静：失去情绪张力
❌ 环境音太吵：干扰情感表达
```

### 4.3 剪辑禁忌

```
❌ 节奏太快：观众来不及感受痛苦
❌ 转场太硬：破坏情绪氛围
❌ 没有慢动作：错过心碎瞬间
```

## 六、虐恋剧节奏模板

### 6.1 单集节奏（3分钟）

```
0:00-0:05  钩子：虐心预告/回忆闪回
0:05-0:30  铺垫：甜蜜回忆/误会加深
0:30-1:00  发展：冲突升级/痛苦加深
1:00-1:30  高潮：虐心时刻/分离/误解
1:30-2:00  转折：真相出现/希望
2:00-2:30  再虐：更大的误会/阻碍
2:30-3:00  结尾：悬念/虐心预告
```

### 6.2 情绪节奏

```
甜蜜：20%
虐心：50%
希望：15%
绝望：15%
```
