# 不动如山 · 军事常识速查手册

## 一、350.3高地地形与阵地构型

### 实际地形
- 标高 350.3 米，汉江南岸独立高地，控制京畿道利川-骊州公路
- 南坡陡（约35°）、北坡缓（约20°）——敌人主攻方向在南坡
- 山顶面积约 0.3 平方公里，可容纳约一个连的环形防御
- 冬季冻土深度 40-60cm，镐头挖不动，只能用炸药

### 志愿军连级防御阵地标准构型
```
第一道堑壕（前沿 150m 外）
  ├─ 散兵坑间隔 8-10m（防一枚迫击炮同时杀伤两人）
  ├─ 机枪掩体：交叉火力覆盖南坡主攻轴线
  └─ 反斜面迫击炮位：北坡挖入式，防直射火力

第二道堑壕（山顶线后 50m）
  ├─ 预备队集结位
  ├─ 连指挥掩蔽部（2m深，原木+冻土覆盖，防105mm直击）
  └─ 弹药囤积点

第三道（北坡反斜面）
  ├─ 伤员收集点
  ├─ 通信线（通往营部/团部）
  └─ 最后的撤退路线 → 汉江冰面
```

### 现有剧本差距
- 剧本中"战壕"被提到76次但没有区分"第一道/第二道/反斜面"
- 没有标注机枪掩体的射界覆盖角度
- 掩蔽部深度、结构未交代

### 弹药消耗现实
- 一个连每日正常消耗：步枪弹 2000发 + 机枪弹 5000发 + 迫击炮弹 30-50发 + 手榴弹 100枚
- 350.3高地实际供应：**远低于此**——补给线被炮火封锁，靠夜间人力背运
- 打到后期（G47-G52），士兵平均弹药：步枪兵 20-30发 / 机枪手 2-3个弹盘

### 现有剧本差距
- 剧本提到"弹药不够"但从未给出具体数字——观众感受不到什么叫"不够"
- 没有标注武器型号，涂金的"机枪"到底是什么机枪？
- 迫击炮（傅国良）的炮弹存量、射击角度未交代

## 三、美军进攻战术（1951年2月·李奇微"磁性战术"期间）

### 标准连级进攻流程
1. **炮火准备**（30min）：105mm/155mm 榴弹炮覆盖 → 阵地翻一遍
2. **坦克抵近**：M4谢尔曼/M26潘兴在 500m 外直射机枪掩体
3. **步兵散兵线推进**：间距 5-8m，两列交替跃进
4. **火力引导员**：携带 SCR-300 电台，呼叫炮兵修正落点（曹玉海打掉的就是他）

### "磁性战术"核心
- 白天进攻、夜间脱离接触（避免志愿军夜战优势）
- 炮火+飞机无限量使用（美军炮弹是志愿军的 50-100 倍）
- 逐步压缩阵地，不追求一次突破

### 现有剧本差距
- 剧本中对美军进攻的描写偏"感觉"——"炮密得像下雨"——缺少具体参数
- 没有交代美军的进攻队形（散兵线间隔、交替跃进）
- 李奇微的G29A/G48A/G65A已有战略层面，但基层的美军士兵是"模糊的影子"

## 四、志愿军防御战术

### 核心防御原则
- **前轻后重**：前沿只放少量观察哨+机枪，主力在第二道堑壕
- **反斜面配置**：迫击炮、预备队藏在北坡（防美军直射火力）
- **短促突击**：趁美军炮火转移时，从反斜面跃出打 3-5 分钟近战，然后撤回
- **三三制**：三人一组，两组交替掩护——但 350.3 后期人不够，用不了

### 现有剧本的战术正确场景
- G47 夜间摸哨 ✅（短促突击）
- G52 曹玉海打掉火力引导员 ✅（战术正确）
- G42 "我们不动" ✅（前轻后重原则的文学表达）

### 现有剧本的战术模糊场景
- G48-G51 大规模进攻：没有交代美军的进攻队形，也没有交代志愿军的火力分配
- 缺少"反斜面"概念——这个地形细节对视觉呈现非常重要

## 五、战场医疗与后勤

### 志愿军连级卫生员配置
- 每连 1-2 名卫生员（孙殿金就是）
- 装备：急救包（吗啡+磺胺粉+绷带+止血带）、水壶、剪刀
- 能做的：止血、包扎、骨折固定、吗啡止痛
- 不能做的：手术、输血、取弹片（需后送到营/团救护所）

### 补给线
- 汉江封冻期（1-2月）：冰面可走，夜间人力背运
- 3月化冰后：补给完全断绝（汉江不能走，没有桥）
- 350.3 打到后期（G52前），每人每天只有一把炒面

### 现有剧本差距
- 孙殿金救助伤员的描写有情感力量但缺少具体医疗动作
- 炒面/水壶提到了但没交代"为什么只有这些"
- 冻伤的医学现实未触及：志愿军大量冻伤减员（脚趾变黑→截肢）

## 六、剧本台词可插入的战术细节（建议清单）

### 可以自然嵌入的对白/旁白
1. G36 方新写花名册时：「七号阵地。一班。没了。」→ 阵地编号系统
2. 曹玉海对涂金：「机枪射界——左边到那颗炸断的树。右边到那块黑石头。」→ 射界标注
3. 方新整理弹药：「波波沙。十七个弹鼓。七十一发一个。一千二百零七。」→ 数弹药 = 数字的力量
4. 傅国良：「最后三发。八十二迫。我留一发给连长。」→ 炮型号+炮弹情感
5. 周二柱：「反斜面上还有人。我看见动了。」→ 反斜面概念
6. 沈德恩数子弹：「七发。七十六毫米。够打死两个人。」→ 弹径

### 不建议插入的
- 长篇战术讲解（破坏节奏）
- 军事术语堆砌（观众听不懂）
- 司令部里面的沙盘推演（已有G4/G8/G14）
