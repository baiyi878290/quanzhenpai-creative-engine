# 竖屏短剧制作指南

## 一、竖屏短剧概述

### 1.1 定义与特点
```
竖屏短剧：专为移动端竖屏（9:16）设计的短篇剧集
特点：
- 单集时长：1-5分钟
- 总集数：10-100集
- 画幅：9:16竖屏
- 节奏：快节奏，每集有高潮
- 付费模式：前3-5集免费，后续付费
- 目标：高完播率+高付费转化
```

### 1.2 热门类型
```
1. 甜宠/霸总（女性向，流量最大）
2. 复仇/逆袭（男女通吃）
3. 穿越/重生（题材新颖）
4. 悬疑/推理（粘性高）
5. 奇幻/异能（视觉冲击）
6. 古装/仙侠（制作要求高）
```

### 1.3 平台适配
```
| 平台 | 画幅 | 时长 | 付费模式 |
|------|------|------|----------|
| 抖音短剧 | 9:16 | 1-3分钟 | IAP+IAA |
| 快手短剧 | 9:16 | 1-3分钟 | IAP+IAA |
| 微信小程序 | 9:16 | 1-5分钟 | IAP |
| 红果短剧 | 9:16 | 1-3分钟 | IAA |
| 河马剧场 | 9:16 | 1-3分钟 | IAP |
```

## 二、竖屏短剧叙事结构

### 2.1 单集结构（1-3分钟）
```
0-5秒：钩子（冲突/悬念/反转/视觉冲击）
5-30秒：铺垫（快速建立情境）
30-90秒：发展（推进冲突/情感）
90-150秒：高潮（情感爆发/反转）
150-180秒：钩子结尾（悬念/期待下一集）
```

### 2.2 剧集结构（10-100集）
```
第1-3集：免费期
- 第1集：建立世界观+核心冲突+钩子
- 第2集：深化冲突+角色关系
- 第3集：第一个小高潮+付费卡点

第4-10集：发展期
- 冲突升级
- 角色成长
- 感情线推进

第11-30集：高潮期
- 多线并进
- 反转密集
- 情感爆发

第31集+：延续期
- 新冲突引入
- 角色弧完善
- 大结局
```

### 2.3 付费点设计
```
原则：
1. 付费点设在情感高潮处（观众最想知道后续）
2. 第3集末尾设第一个付费卡点
3. 每5-8集设一个大付费卡点
4. 付费前一集必须有强钩子

常见付费卡点：
- 身份揭示（"其实我是..."）
- 关系转折（表白/分手/背叛）
- 悬念留白（凶手是...）
- 反转铺垫（真相即将揭晓）
```

## 三、竖屏短剧镜头语言

### 3.1 竖屏构图原则
```
1. 人物居中：竖屏以人物为核心
2. 上留天：头顶留白1/5
3. 下留地：脚下留白1/8
4. 避免横屏思维：不要左右对称构图
5. 利用竖屏优势：楼梯/电梯/走廊等纵向场景
```

### 3.2 常用景别
```
| 景别 | 用途 | 竖屏特点 |
|------|------|----------|
| 特写 | 情绪表达 | 占屏幕70%以上 |
| 近景 | 对话/表情 | 头部+肩膀 |
| 中景 | 动作/互动 | 腰部以上 |
| 全身 | 走路/动作 | 完整人物 |
| 远景 | 环境交代 | 少用，竖屏不适合 |
```

### 3.3 运镜技巧
```
1. 推镜头：情绪强化（特写→大特写）
2. 拉镜头：揭示关系（特写→中景）
3. 跟镜头：跟随角色（走路/奔跑）
4. 甩镜头：快速转场（适合快节奏）
5. 固定镜头：对话/情感戏
6. 手持镜头：增加真实感/紧张感
```

## 四、竖屏短剧制作流程

### 4.1 前期
```
1. 选题：市场调研+类型选择
2. 剧本：10-30天完成全部剧本
3. 分镜：竖屏分镜（9:16画幅）
4. 选角：适配竖屏的演员（面部表情丰富）
5. 场地：竖屏适配的场景设计
```

### 4.2 中期
```
1. 拍摄：竖屏拍摄（手机/竖屏监视器）
2. 日程：10-20天拍完全集
3. 效率：每天拍5-10集
4. 注意：竖屏构图实时监看
```

### 4.3 后期
```
1. 剪辑：快节奏，每集独立成篇
2. 调色：明亮/饱和/对比强
3. 音效：BGM+音效+配音
4. 字幕：大字号，位置居中
5. 包装：片头/片尾/角标
```

## 五、AI辅助竖屏短剧制作

### 5.1 AI工具链
```
剧本阶段：AI剧本分析+分镜拆解
生成阶段：Seedance 2.0生成画面
配音阶段：ElevenLabs/小米TTS配音
配乐阶段：Suno/Udio生成BGM
后期阶段：FFmpeg合成+调色
```

### 5.2 一键成片流程
```
1. 输入剧本文本
2. AI分析生成分镜
3. 逐镜头AI画面生成
4. AI配音+配乐
5. 自动合成成片
6. 人工审核+微调
```

### 5.3 成本对比
```
传统制作：
- 演员：500-2000元/集
- 场地：300-1000元/天
- 设备：500-2000元/天
- 后期：300-800元/集
- 总计：约1000-4000元/集

AI辅助制作：
- AI生成：100-500元/集
- 配音：50-200元/集
- 配乐：50-100元/集
- 总计：约200-800元/集
- 成本降低：70-80%
```
