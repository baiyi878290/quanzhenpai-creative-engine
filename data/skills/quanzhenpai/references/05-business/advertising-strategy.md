# 投流策略指南

## 一、DOU+投放指南

### 1.1 DOU+基础

```
DOU+是什么：
- 抖音官方付费推广工具
- 帮助视频获得更多曝光
- 精准投放目标用户

DOU+预算：
- 最低50元起
- 按播放量计费
- 约100元=5000-10000播放
```

### 1.2 DOU+投放策略

**测试期（第1-3天）：**
```
预算：100-300元/条
目标：播放量
时长：24小时
定向：系统智能推荐

操作：
1. 视频发布2小时后投放
2. 观察完播率和互动率
3. 数据好则追加投放
4. 数据差则停止投放
```

**放量期（第4-7天）：**
```
预算：1000-5000元/条
目标：点赞评论
时长：48-72小时
定向：自定义人群

操作：
1. 选择目标人群
2. 设置投放时段
3. 每天检查数据
4. 及时调整出价
```

**爆发期（第7天+）：**
```
预算：5000-20000元/条
目标：粉丝增长/转化
时长：7天
定向：相似账号粉丝

操作：
1. 定向相似账号
2. 定向竞品粉丝
3. 大预算投放
4. 配合直播/活动
```

### 1.3 DOU+定向设置

```
基础定向：
├── 性别：男/女/不限
├── 年龄：18-23/24-30/31-40/41-49/50+
├── 地域：全国/指定省市
└── 兴趣：系统推荐/自定义

高级定向：
├── 行为兴趣：搜索/浏览/购买
├── 达人相似：竞品账号粉丝
├── 自定义人群：已有粉丝/潜在用户
└── 智能推荐：系统自动优化
```

### 1.4 DOU+数据监控

| 指标 | 参考值 | 优化方向 |
|------|--------|----------|
| 完播率 | >30% | 优化内容质量 |
| 点赞率 | >5% | 增强情感共鸣 |
| 评论率 | >1% | 增加话题性 |
| 转发率 | >0.5% | 增强传播性 |
| 关注率 | >1% | 优化人设/系列感 |
| CPC | <0.1元 | 优化投放策略 |

## 二、千川投放指南

### 2.1 千川基础

```
千川是什么：
- 巨量引擎旗下的广告投放平台
- 适合电商带货和直播引流
- 支持短视频带货和直播带货

千川特点：
- 精准定向
- 智能出价
- 多种转化目标
- 数据追踪完善
```

### 2.2 千川投放策略

**短视频带货：**
```
预算：500-2000元/天
目标：商品购买/下单
出价方式：OCPM
素材：3-5条轮换

操作：
1. 准备3-5条优质素材
2. 设置目标人群
3. 选择出价方式
4. 每天检查数据
5. 优化素材和出价
```

**直播引流：**
```
预算：1000-5000元/天
目标：进入直播间/直播间下单
出价方式：OCPM
素材：直播切片

操作：
1. 准备直播切片素材
2. 设置引流时段
3. 直播前1-2小时投放
4. 直播中追加投放
5. 监控直播间数据
```

### 2.3 千川出价策略

```
出价公式：
目标CPA = 商品利润 × 目标ROI

例如：
商品利润：50元
目标ROI：2
目标CPA：25元

出价范围：
- 保守出价：目标CPA × 0.8
- 适中出价：目标CPA × 1.0
- 激进出价：目标CPA × 1.2
```

### 2.4 千川素材优化

```
素材类型：
1. 产品展示（15-30秒）
2. 使用场景（15-30秒）
3. 效果对比（15-30秒）
4. 用户证言（15-30秒）
5. 直播切片（15-30秒）

素材结构：
0-3秒：钩子（吸引注意）
3-15秒：产品展示/痛点
15-25秒：效果/证言
25-30秒：行动号召
```

## 三、ROI计算与优化

### 3.1 ROI基础

```
ROI = 收入 / 投入

例如：
投入：1000元（广告费）
收入：3000元（销售额）
ROI = 3000 / 1000 = 3

ROI>1：盈利
ROI=1：持平
ROI<1：亏损
```

### 3.2 目标ROI设定

| 行业 | 目标ROI | 说明 |
|------|---------|------|
| 服装 | 2-3 | 利润率中等 |
| 美妆 | 3-5 | 利润率高 |
| 食品 | 1.5-2 | 利润率低 |
| 数码 | 2-3 | 利润率中等 |
| 家居 | 3-5 | 利润率高 |

### 3.3 ROI优化方法

```
1. 提高转化率
   - 优化素材
   - 优化落地页
   - 优化产品描述

2. 提高客单价
   - 组合销售
   - 满减活动
   - 高端产品

3. 降低成本
   - 优化出价
   - 优化定向
   - 提高素材质量

4. 提高复购
   - 会员体系
   - 定期上新
   - 私域运营
```

## 六、投放避坑指南

### 6.1 常见错误

```
❌ 预算分配不均
❌ 素材单一
❌ 定向过窄
❌ 出价过高/过低
❌ 不看数据
❌ 频繁调整
```

### 6.2 优化建议

```
✅ 预算分散投放
✅ 素材多版本测试
✅ 定向适度放宽
✅ 出价根据数据调整
✅ 每天看数据
✅ 稳定投放周期
```
