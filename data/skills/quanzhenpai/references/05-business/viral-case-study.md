# 爆款拆解库

## 一、爆款公式

### 1.1 通用爆款公式

```
爆款 = 强钩子 × 高密度爽点 × 情绪共鸣 × 系列粘性

强钩子：前3秒决定生死
高密度爽点：每30秒一个刺激
情绪共鸣：触动观众内心
系列粘性：让人想看下一集
```

### 1.2 竖屏短剧爆款公式

```
竖屏短剧 = 冲突开局 × 反转不断 × 甜虐交替 × 悬念结尾

冲突开局：第一集建立核心冲突
反转不断：每集至少1个反转
甜虐交替：给糖又虐心
悬念结尾：让人欲罢不能
```

## 二、爆款案例拆解

### 2.1 甜宠爆款案例

**案例1：《霸总的替身新娘》**

```
数据：
- 播放量：2.3亿
- 点赞：890万
- 评论：45万
- 完播率：45%

成功要素：
1. 强钩子：替身梗，身份反转
2. 人设：霸道总裁+倔强女主
3. 爽点：打脸+撒糖+虐心
4. 节奏：每集3个以上爽点
5. 悬念：身份何时揭晓

可学习点：
- 替身/身份反转梗永不过时
- 霸总人设经久不衰
- 甜虐交替保持节奏
```

## 五、爆款检查清单

### 5.1 内容检查

```
□ 前3秒是否有强钩子？
□ 是否有核心冲突？
□ 是否有爽点/泪点？
□ 是否有反转？
□ 结尾是否有悬念？
```

### 5.2 技术检查

```
□ 画面是否清晰？
□ 节奏是否紧凑？
□ 配乐是否合适？
□ 字幕是否清晰？
□ 时长是否合适？
```

### 5.3 商业检查

```
□ 是否适合目标平台？
□ 是否有付费点？
□ 是否有传播性？
□ 是否有系列性？
□ 是否有记忆点？
```
