# 镜头调度设计

## 一、镜头调度基础

### 1.1 什么是镜头调度

```
镜头调度 = 人物走位 + 机位设置 + 运镜配合

核心思想：
通过人物和镜头的配合
创造视觉张力和叙事节奏
```

### 1.2 镜头调度三要素

| 要素 | 说明 | 设计要点 |
|------|------|----------|
| 人物走位 | 人物在空间中的移动 | 位置变化=关系变化 |
| 机位设置 | 摄像机的位置 | 角度=权力关系 |
| 运镜配合 | 摄像机的运动 | 运动=情绪节奏 |

---

## 二、人物走位设计

### 2.1 位置与关系

| 位置关系 | 心理暗示 | 适用场景 |
|----------|----------|----------|
| 面对面 | 对峙、对话 | 冲突、交流 |
| 并肩 | 同伴、亲密 | 日常、同行 |
| 前后 | 引导、跟随 | 走路、追逐 |
| 对角线 | 张力、不安 | 冲突、悬疑 |
| 远离 | 疏离、分离 | 告别、分手 |
| 靠近 | 亲密、压迫 | 甜蜜、威胁 |

### 2.2 走位设计模板

**对峙走位：**
```
初始：两人面对面，距离2米
发展：一方前进，另一方后退
高潮：距离缩短到0.5米
释放：一方转身离开
```

**甜蜜走位：**
```
初始：两人并肩行走
发展：手部靠近
高潮：牵手/拥抱
释放：依偎在一起
```

**追逐走位：**
```
初始：追逐者在后，被追者在前
发展：距离缩短
高潮：追上/抓住
释放：被追者停下/反击
```

## 三、机位设置设计

### 3.1 机位与权力

| 机位 | 权力关系 | 心理效果 |
|------|----------|----------|
| 正面平视 | 平等 | 客观、对话 |
| 正面仰拍 | 被摄者强势 | 威严、霸气 |
| 正面俯拍 | 被摄者弱势 | 渺小、可怜 |
| 侧面 | 中立 | 观察、叙事 |
| 背面 | 神秘 | 神秘、期待 |
| 过肩 | 第三方视角 | 代入、参与 |

### 3.2 机位切换规则

```
180度轴线规则：
- 保持在轴线同侧
- 避免跳轴
- 保持空间关系清晰

机位切换逻辑：
A看B → 切B → 切A反应
A说话 → 切B反应 → 切A继续
```

## 四、运镜配合设计

### 4.1 运镜与动作配合

| 动作 | 运镜 | 效果 |
|------|------|------|
| 走近 | 缓慢推进 | 期待、紧张 |
| 走远 | 缓慢拉远 | 疏离、释放 |
| 转身 | 跟随旋转 | 迷失、眩晕 |
| 奔跑 | 跟随拍摄 | 紧迫、沉浸 |
| 对峙 | 固定/正反打 | 紧张、张力 |

### 4.2 运镜与情绪配合

| 情绪 | 运镜 | 效果 |
|------|------|------|
| 紧张 | 快速推进 | 冲击、震惊 |
| 悲伤 | 缓慢拉远 | 疏离、释放 |
| 浪漫 | 缓慢推进 | 聚焦、亲密 |
| 愤怒 | 快速推进+晃动 | 冲击、不稳定 |
| 回忆 | 缓慢推进+柔焦 | 梦幻、怀旧 |

## 五、镜头调度实战

### 5.1 对峙场景调度

```
场景：两人在办公室对峙

镜头1：全景，两人面对面
- 机位：正面平视
- 人物：相距2米

镜头2：A的过肩镜头看B
- 机位：A背后
- 人物：B正面

镜头3：B的过肩镜头看A
- 机位：B背后
- 人物：A正面

镜头4：特写A的表情
- 机位：正面
- 运镜：缓慢推进

镜头5：特写B的表情
- 机位：正面
- 运镜：缓慢推进

镜头6：全景，A转身离开
- 机位：正面
- 运镜：固定
```

### 5.2 甜蜜场景调度

```
场景：两人在咖啡厅约会

镜头1：中景，两人面对面
- 机位：侧面
- 人物：相距1米

镜头2：特写A看B
- 机位：A视角
- 人物：B微笑

镜头3：特写B看A
- 机位：B视角
- 人物：A害羞

镜头4：近景，A伸手
- 机位：侧面
- 人物：A手伸向B

镜头5：特写，手牵手
- 机位：俯拍
- 人物：手部特写

镜头6：近景，两人相视而笑
- 机位：正面
- 人物：两人并排
```

### 5.3 追逐场景调度

```
场景：A追B

镜头1：全景，B在前面跑
- 机位：B后面
- 运镜：跟随

镜头2：全景，A在后面追
- 机位：A后面
- 运镜：跟随

镜头3：中景，B回头看
- 机位：B侧面
- 运镜：固定

镜头4：中景，A伸手
- 机位：A侧面
- 运镜：固定

镜头5：全景，A抓住B
- 机位：侧面
- 运镜：固定

镜头6：近景，两人对视
- 机位：正面
- 运镜：固定
```
