# 调色原理指南

## 一、调色基础

### 1.1 什么是调色

```
调色 = 色彩校正 + 色彩风格化

色彩校正：让画面颜色准确
色彩风格化：让画面有情绪
```

### 1.2 调色工具

| 工具 | 特点 | 适用 |
|------|------|------|
| DaVinci Resolve | 专业、免费版够用 | 专业制作 |
| 剪映 | 简单、内置滤镜 | 快速处理 |
| Premiere Pro | 集成、插件多 | 综合制作 |
| LUT预设 | 一键调色 | 批量处理 |

## 二、调色流程

### 2.1 标准调色流程

```
Step 1: 白平衡校正
- 修正色偏
- 让白色真正是白色

Step 2: 曝光校正
- 调整亮度
- 让画面不过曝/欠曝

Step 3: 对比度调整
- 调整明暗对比
- 让画面有层次

Step 4: 饱和度调整
- 调整颜色纯度
- 让画面有活力

Step 5: 色彩风格化
- 添加色调
- 创造情绪

Step 6: 细节调整
- 锐化
- 降噪
```

### 2.2 竖屏短剧调色流程

```
1. 统一素材
- 所有素材白平衡一致
- 所有素材曝光一致

2. 基础校正
- 白平衡
- 曝光
- 对比度

3. 风格化
- 应用LUT
- 调整色调
- 统一风格

4. 局部调整
- 皮肤美化
- 天空调整
- 焦点强调
```

## 三、LUT应用

### 3.1 什么是LUT

```
LUT = Look Up Table（查找表）

作用：
- 一键应用色彩风格
- 统一全片色调
- 快速调色
```

### 3.2 常用LUT

| LUT | 风格 | 适用场景 |
|-----|------|----------|
| Kodak 2383 | 电影胶片 | 情感剧、文艺片 |
| Fuji Pro 400H | 清新自然 | 甜宠、日常 |
| Cyberpunk Teal&Orange | 赛博朋克 | 科幻、悬疑 |
| Wong Kar-wai Style | 复古港风 | 怀旧、爱情 |
| Black & White | 黑白 | 回忆、艺术 |

### 3.3 LUT应用技巧

```
1. 适度使用
- LUT强度50-80%
- 避免过度

2. 配合调整
- LUT后再微调
- 适配具体素材

3. 统一应用
- 全片使用相同LUT
- 保持风格一致
```

## 四、情绪调色

### 4.1 甜蜜调色

```
基础：
- 色温：偏暖（+10）
- 色调：偏洋红（+5）
- 饱和度：+15%
- 对比度：-10%

HSL：
- 橙色：饱和度+20，明度+10
- 红色：饱和度+10
- 蓝色：饱和度-20

曲线：
- 提亮暗部
- 压暗高光
- 增加对比
```

### 4.2 悲伤调色

```
基础：
- 色温：偏冷（-10）
- 色调：偏青（+5）
- 饱和度：-20%
- 对比度：+10%

HSL：
- 蓝色：饱和度+20
- 橙色：饱和度-30
- 绿色：饱和度-20

曲线：
- 压暗暗部
- 提亮高光
- 增加对比
```

### 4.3 愤怒调色

```
基础：
- 色温：中性
- 色调：偏红（+10）
- 饱和度：+10%
- 对比度：+20%

HSL：
- 红色：饱和度+30
- 橙色：饱和度+10
- 蓝色：饱和度-20

曲线：
- 增加对比
- 压暗暗部
- 提亮高光
```

### 4.4 悬疑调色

```
基础：
- 色温：偏冷（-15）
- 色调：偏青（+10）
- 饱和度：-30%
- 对比度：+20%

HSL：
- 蓝色：饱和度+20
- 绿色：饱和度-20
- 橙色：饱和度-30

曲线：
- 压暗暗部
- 压暗中间调
- 提亮高光
```

## 六、DaVinci调色节点

### 6.1 基础节点结构

```
节点1：白平衡校正
节点2：曝光校正
节点3：对比度调整
节点4：饱和度调整
节点5：色彩风格化
节点6：局部调整
```

### 6.2 常用节点操作

```
白平衡：
- 使用白平衡工具
- 点击白色区域
- 自动校正

曝光：
- 调整Lift（暗部）
- 调整Gamma（中间调）
- 调整Gain（高光）

对比度：
- 调整Contrast
- 调整Pivot

饱和度：
- 调整Saturation
- 调整Vibrance
```
