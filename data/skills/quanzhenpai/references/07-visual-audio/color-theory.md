# 色彩理论指南

## 一、色彩基础

### 1.1 色彩三要素

| 要素 | 说明 | 影响 |
|------|------|------|
| 色相 | 颜色的种类（红/蓝/绿） | 情绪基调 |
| 饱和度 | 颜色的纯度 | 活力程度 |
| 明度 | 颜色的明暗 | 轻松/沉重 |

### 1.2 色温

| 色温 | 情绪 | 适用场景 |
|------|------|----------|
| 暖色（3000K） | 温馨、浪漫 | 甜蜜、日常 |
| 中性（5000K） | 自然、客观 | 对话、叙事 |
| 冷色（7000K） | 忧郁、疏离 | 悲伤、悬疑 |

## 四、色彩搭配原则

### 4.1 60-30-10法则

```
主色：60%（背景、大面积）
辅色：30%（次要元素）
点缀色：10%（强调元素）

例如：
主色：白色（60%）
辅色：粉色（30%）
点缀色：红色（10%）
```

### 4.2 色彩和谐

```
同色系：同一色相的不同明度/饱和度
类似色：色轮上相邻的颜色
互补色：色轮上相对的颜色
三色组：色轮上等距的三种颜色
```

### 4.3 色彩对比

```
明暗对比：明色+暗色
冷暖对比：冷色+暖色
饱和对比：高饱和+低饱和
色相对比：互补色
```
