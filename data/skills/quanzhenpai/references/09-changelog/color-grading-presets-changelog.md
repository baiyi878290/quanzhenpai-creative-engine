### 调色预设使用指南

#### DaVinci Resolve
```
1. 导入预设
   - 打开Color页面
   - 右键节点 → Import
   - 选择.cube或.3dl文件

2. 应用预设
   - 拖拽预设到节点
   - 调整强度（50-80%）
   - 微调参数

3. 保存预设
   - 调整满意后
   - 右键节点 → Export
   - 保存为.cube文件
```

#### 剪映
```
1. 使用滤镜
   - 选择对应滤镜
   - 调整强度
   - 应用到片段

2. 自定义调色
   - 调整基础参数
   - 调整HSL
   - 调整曲线
   - 保存为自定义滤镜
```
