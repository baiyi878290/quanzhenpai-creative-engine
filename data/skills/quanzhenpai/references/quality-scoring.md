# 成片评分体系

## 一、10维度评分标准

### 1.1 评分维度

| 维度 | 权重 | 评分项 | 满分 |
|------|------|--------|------|
| 画面质量 | 15% | 清晰度、稳定性、无伪影 | 10 |
| 角色一致性 | 15% | 外观、服装、发型一致 | 10 |
| 场景一致性 | 10% | 布局、光影、色调一致 | 10 |
| 动作自然度 | 10% | 动作流畅、无变形 | 10 |
| 情绪表达 | 15% | 表情准确、情绪递进 | 10 |
| 节奏控制 | 10% | 快慢适当、剪辑流畅 | 10 |
| 声音设计 | 10% | 配音、配乐、音效协调 | 10 |
| 叙事逻辑 | 5% | 故事清晰、无漏洞 | 10 |
| 情感感染 | 5% | 能引发观众情绪 | 10 |
| 商业潜力 | 5% | 完播率、付费意愿 | 10 |

### 1.2 评分等级

| 分数 | 等级 | 描述 | 平台定位 |
|------|------|------|----------|
| 90-100 | S级 | 院线/平台精品 | 头部推荐 |
| 80-89 | A级 | 平台推荐级别 | 首页推荐 |
| 70-79 | B级 | 可上线级别 | 普通上线 |
| 60-69 | C级 | 需要优化 | 测试上线 |
| <60 | D级 | 不合格 | 不建议上线 |

## 二、各维度评分细则

### 2.1 画面质量（15%）

| 分数 | 标准 |
|------|------|
| 10 | 4K清晰，无任何瑕疵，专业级画质 |
| 8-9 | 1080p清晰，基本无瑕疵 |
| 6-7 | 720p清晰，有轻微瑕疵 |
| 4-5 | 480p，有明显瑕疵 |
| 1-3 | 模糊，有严重瑕疵 |

**检查项：**
```
□ 分辨率是否达标？
□ 是否有黑帧/花帧？
□ 是否有闪烁/抖动？
□ 是否有压缩伪影？
□ 是否有噪点？
□ 是否有色彩断层？
```

### 2.2 角色一致性（15%）

| 分数 | 标准 |
|------|------|
| 10 | 所有镜头角色完全一致，无任何偏差 |
| 8-9 | 基本一致，有极微小偏差 |
| 6-7 | 大部分一致，有少量偏差 |
| 4-5 | 部分一致，有明显偏差 |
| 1-3 | 严重不一致，角色变形 |

**检查项：**
```
□ 发型是否一致？
□ 服装是否一致？
□ 五官是否一致？
□ 体型是否一致？
□ 肤色是否一致？
□ 配饰是否一致？
```

### 2.3 场景一致性（10%）

| 分数 | 标准 |
|------|------|
| 10 | 同一场景完全一致，无任何偏差 |
| 8-9 | 基本一致，有极微小偏差 |
| 6-7 | 大部分一致，有少量偏差 |
| 4-5 | 部分一致，有明显偏差 |
| 1-3 | 严重不一致，场景混乱 |

**检查项：**
```
□ 布局是否一致？
□ 家具位置是否一致？
□ 光影方向是否一致？
□ 色调是否一致？
□ 装饰品是否一致？
```

### 2.4 动作自然度（10%）

| 分数 | 标准 |
|------|------|
| 10 | 动作完全自然，流畅无变形 |
| 8-9 | 基本自然，有极微小不自然 |
| 6-7 | 大部分自然，有少量不自然 |
| 4-5 | 部分自然，有明显不自然 |
| 1-3 | 严重不自然，动作变形 |

**检查项：**
```
□ 动作是否流畅？
□ 是否有变形？
□ 是否有卡顿？
□ 速度是否正常？
□ 肢体是否协调？
```

### 2.5 情绪表达（15%）

| 分数 | 标准 |
|------|------|
| 10 | 情绪层次丰富，递进自然，感染力强 |
| 8-9 | 情绪表达清晰，有递进 |
| 6-7 | 情绪基本清晰，递进不明显 |
| 4-5 | 情绪模糊，缺乏递进 |
| 1-3 | 情绪错误，表达混乱 |

**检查项：**
```
□ 表情是否准确？
□ 眼神是否有情绪？
□ 是否有微表情？
□ 情绪是否有递进？
□ 是否有情绪层次？
```

### 2.6 节奏控制（10%）

| 分数 | 标准 |
|------|------|
| 10 | 节奏完美，与情绪完美匹配 |
| 8-9 | 节奏良好，基本匹配情绪 |
| 6-7 | 节奏基本合适，有少量不匹配 |
| 4-5 | 节奏有明显问题 |
| 1-3 | 节奏混乱，严重影响观看 |

**检查项：**
```
□ 快慢是否适当？
□ 剪辑是否流畅？
□ 转场是否自然？
□ 是否有跳切？
□ 时长是否合理？
```

### 2.7 声音设计（10%）

| 分数 | 标准 |
|------|------|
| 10 | 声音设计完美，与画面完美配合 |
| 8-9 | 声音良好，基本配合画面 |
| 6-7 | 声音基本合适，有少量问题 |
| 4-5 | 声音有明显问题 |
| 1-3 | 声音混乱，严重影响观看 |

**检查项：**
```
□ 配音是否清晰？
□ 配乐是否合适？
□ 音效是否准确？
□ 音量是否平衡？
□ 是否有噪音？
```

### 2.8 叙事逻辑（5%）

| 分数 | 标准 |
|------|------|
| 10 | 故事清晰，逻辑严密，无漏洞 |
| 8-9 | 故事清晰，基本无漏洞 |
| 6-7 | 故事基本清晰，有少量漏洞 |
| 4-5 | 故事有明显漏洞 |
| 1-3 | 故事混乱，无法理解 |

**检查项：**
```
□ 故事是否清晰？
□ 时间线是否合理？
□ 人物动机是否合理？
□ 是否有逻辑漏洞？
□ 是否有信息缺失？
```

### 2.9 情感感染（5%）

| 分数 | 标准 |
|------|------|
| 10 | 能强烈引发观众情绪，有泪点/笑点/爽点 |
| 8-9 | 能引发观众情绪 |
| 6-7 | 部分场景能引发情绪 |
| 4-5 | 情感感染力弱 |
| 1-3 | 无情感感染力 |

**检查项：**
```
□ 是否有泪点？
□ 是否有笑点？
□ 是否有爽点？
□ 是否有心动瞬间？
□ 观众是否会有情绪反应？
```

### 2.10 商业潜力（5%）

| 分数 | 标准 |
|------|------|
| 10 | 有爆款潜力，高完播率，强付费意愿 |
| 8-9 | 有较好商业潜力 |
| 6-7 | 商业潜力一般 |
| 4-5 | 商业潜力较弱 |
| 1-3 | 无商业潜力 |

**检查项：**
```

□ 是否有爽点/泪点？
□ 结尾是否有悬念？
□ 是否有付费点？
□ 是否适合目标平台？
```

## 三、观众心理曲线

### 3.1 3分钟短剧理想曲线

```
情绪
  ↑
高│        ╱╲            ╱╲
  │       ╱  ╲          ╱  ╲
中│  ╱╲  ╱    ╲    ╱╲  ╱    ╲
  │ ╱  ╲╱      ╲  ╱  ╲╱      ╲
低│╱              ╲╱            ╲
  └──────────────────────────────→ 时间
   0:00  0:30  1:00  1:30  2:00  2:30  3:00
   钩子  铺垫  高潮1 转折  高潮2 结尾
```

### 3.2 关键节点

```
0:00-0:03：钩子（决定是否继续看）
0:30：第一个爽点（决定是否看完）
1:00-1:30：情绪高潮（核心吸引力）
2:00：反转/揭秘（增加粘性）
2:30-3:00：结尾钩子（决定是否看下一集）
```

### 3.3 完播率影响因素

| 时间点 | 影响因素 | 优化建议 |
|--------|----------|----------|
| 0-3秒 | 钩子强度 | 强冲突/悬念/反转 |
| 3-10秒 | 入戏速度 | 快速建立冲突 |
| 10-30秒 | 第一个爽点 | 及时给出刺激 |
| 30-60秒 | 情绪维持 | 持续情绪刺激 |
| 60秒+ | 结尾钩子 | 留悬念给下集 |

## 五、A/B测试方法

### 5.1 测试维度

```
1. 钩子测试：不同开场对完播率的影响
2. 节奏测试：不同剪辑节奏对观看体验的影响
3. 色调测试：不同色调对情绪的影响
4. 配乐测试：不同配乐对情绪的影响
5. 标题测试：不同标题对点击率的影响
```

### 5.2 测试方法

```
1. 制作两个版本（A和B）
2. 只改变一个变量
3. 小范围测试
4. 收集数据
5. 分析结果
6. 选择更优版本
```

### 5.3 测试指标

| 测试项 | 指标 | 工具 |
|--------|------|------|
| 钩子 | 3秒完播率 | 平台数据 |
| 节奏 | 平均观看时长 | 平台数据 |
| 色调 | 点赞率 | 平台数据 |
| 配乐 | 评论情感分析 | 人工分析 |
| 标题 | 点击率 | 平台数据 |

## 七、质量提升建议

### 7.1 画面质量提升

```
1. 使用高清素材
2. 后期超分辨率
3. 稳定处理
4. 降噪处理
5. 调色统一
```

### 7.2 角色一致性提升

```
1. 使用角色参考图
2. 核心词锁定
3. IP-Adapter人脸锁定
4. 同一场景使用相同参考图
```

### 7.3 情绪表达提升

```
1. 学习情绪层次理论
2. 使用微表情
3. 情绪递进设计
4. 反应镜头强化
```

### 7.4 节奏控制提升

```
1. 学习剪辑节奏学
2. 设计情绪曲线
3. 合理使用转场
4. 避免跳切
```
